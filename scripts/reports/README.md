# 报告生成 (Reports)

生成和合并回测报告

## 脚本

- `generate_quarterly_report.py` - 生成季度报告
- `generate_15min_report.py` - 生成15分钟报告
- `reports_all_tf.py` - 所有时间框架报告
- `reports_june.py` - 六月报告
- `merge_reports.py` - 合并多个报告

## 使用示例

```bash
python scripts/reports/generate_quarterly_report.py
python scripts/reports/merge_reports.py
```

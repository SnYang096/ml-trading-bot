# 分析工具 (Analysis)

模型和数据分析工具

## 脚本

- `compare_models.py` - 对比不同模型
- `drift_analysis.py` / `drift_analysis_simple.py` - 概念漂移分析
- `monthly_drift.py` - 月度漂移分析
- `analyze_normalization_effect.py` - 归一化效果分析
- `analyze_vectorbot_results.py` - VectorBT结果分析

## 使用示例

```bash
# 对比模型
python scripts/analysis/compare_models.py

# 漂移分析
python scripts/analysis/drift_analysis.py
```

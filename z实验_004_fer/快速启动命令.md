# FER 策略训练快速启动

## 命令速查表（复制即用）

### Step 1: 构建 Feature Store

```bash
cd /home/yin/trading/ml_trading_bot

mlbot feature-store build --no-docker \
  --config config/strategies/fer \
  --symbols BTCUSDT,ETHUSDT,BNBUSDT,SOLUSDT,XRPUSDT,ADAUSDT \
  --timeframe 240T \
  --start-date 2023-01-01 --end-date 2026-01-01 \
  --warmup-months 6
```

### Step 2: Gate 训练

```bash
# 训练
mlbot train final --no-docker \
  --config config/strategies/fer \
  --features config/strategies/fer/features_gate.yaml \
  --labels config/strategies/fer/labels_rr_extreme.yaml \
  --symbol BTCUSDT,ETHUSDT,BNBUSDT,SOLUSDT,XRPUSDT,ADAUSDT \
  --timeframe 240T --data-path data/parquet_data \
  --start-date 2023-01-01 --end-date 2026-01-01 \
  --holdout-start-date 2024-05-01 --holdout-end-date 2026-01-01 --seed 42

# 记录训练输出目录
GATE_RESULT_DIR="results/train_final_<timestamp>_rr_extreme/fer"

# Gate 优化
mlbot gate apply-archetype \
  --logs ${GATE_RESULT_DIR}/predictions.parquet --strategy fer

python scripts/optimize_gate_unified.py --strategy fer \
  --logs ${GATE_RESULT_DIR}/logs_gated.parquet \
  --output ${GATE_RESULT_DIR}/gate_optimization.json
```

### Step 3: Evidence 训练

```bash
# 训练
mlbot train final --no-docker \
  --config config/strategies/fer \
  --features config/strategies/fer/features_evidence.yaml \
  --labels config/strategies/fer/labels_return_tree.yaml \
  --symbol BTCUSDT,ETHUSDT,BNBUSDT,SOLUSDT,XRPUSDT,ADAUSDT \
  --timeframe 240T --data-path data/parquet_data \
  --start-date 2023-01-01 --end-date 2026-01-01 \
  --holdout-start-date 2024-05-01 --holdout-end-date 2026-01-01 --seed 42

EVIDENCE_RESULT_DIR="results/train_final_<timestamp>_return_tree/fer"

# Evidence 优化
python scripts/optimize_evidence_plateau.py --strategy fer \
  --logs ${EVIDENCE_RESULT_DIR}/logs_gated.parquet \
  --output ${EVIDENCE_RESULT_DIR}/evidence_optimization.json
```

### Step 4: Entry Filter 优化

```bash
python scripts/optimize_entry_filter_plateau.py \
  --logs ${EVIDENCE_RESULT_DIR}/predictions.parquet --strategy fer
```

### Step 5: Execution 优化

```bash
python scripts/optimize_execution_grid.py \
  --logs ${EVIDENCE_RESULT_DIR}/logs_gated.parquet --strategy fer \
  --output ${EVIDENCE_RESULT_DIR}/execution_grid.json
```

### Step 6: PCM 联合回测

```bash
# 单独回测 FER
python scripts/backtest_execution_layer.py \
  --logs ${EVIDENCE_RESULT_DIR}/predictions.parquet \
  --strategy fer

# FER + BPC + ME PCM 联合回测
python scripts/backtest_execution_layer.py \
  --pcm fer:${EVIDENCE_RESULT_DIR}/predictions.parquet \
       bpc:results/<bpc_result_dir>/predictions.parquet \
       me:results/<me_result_dir>/predictions.parquet \
  --quantile-train-start 2025-02-01 \
  --quantile-train-end 2025-08-01
```

## 关键注意事项

### FER 特定验证

1. **Impulse 识别验证**：
   - 检查特征是否能有效识别"单边impulse"存在
   - 核心特征：impulse_failure_score, time_since_impulse

2. **推进效率验证**：
   - price_delta_efficiency 是否为Top重要特征
   - 效率下降是否与反转成功相关

3. **资金吸收验证**：
   - aggressor_absorption_ratio 是否有效
   - trapped_longs_ratio 是否有信号

4. **PCM 仲裁优先级验证**：
   - 确认 FER = 0（最高优先级）
   - 测试 FER 与 ME/BPC 冲突时的仲裁结果

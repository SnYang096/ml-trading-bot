# CRF — Consolidation Range Fade

CRF 在"已确认的盘整区间"内做双向波段：

- **做多**：价格跌到盒子底部（`box_pos_120 <= 0.15`）→ 做多到对边
- **做空**：价格涨到盒子顶部（`box_pos_120 >= 0.85`）→ 做空到对边

前提：`box_stability_120 >= 0.70` 保证"确实是盒子"；`box_width_pct_120 >= 0.04` 确保盒子够宽，值得一做。

## 与已有策略的关系

| 策略 | 语义 | 方向 | 场景 |
|---|---|---|---|
| SRB | 从盒子里突破成功 → 顺势追 | 突破方向 | 动量行情 |
| BPC | 突破后回踩不破结构 → 延续 | 趋势方向 | 趋势延续 |
| ME | 压缩后扩张启动 | 扩张方向 | 动量启动 |
| **CRF** | **盒子里来回扫 → 反向吃均值回归** | **双向** | **盘整期** |

CRF 是前三个策略**失效窗口**的 alpha 来源：当 SRB/BPC/ME 都沉默时（盒子里没突破，没回踩后延续，没扩张），CRF 可以稳定赚到震荡钱。

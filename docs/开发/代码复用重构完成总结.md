## 📝 使用说明

### 启用 breakeven stop

在 `backtest.yaml` 中：
```yaml
rr:
  use_breakeven_stop: true   # 启用保本止损
```

### 从配置读取参数

`model-comparison` 现在会自动从策略配置中读取：
- `stop_loss_r`: 从 `backtest.yaml` 的 `rr.stop_loss_r` 或 `labels.yaml` 的 `generator.params.stop_loss_r`
- `take_profit_r`: 从 `backtest.yaml` 的 `rr.take_profit_r` 或 `labels.yaml` 的 `generator.params.take_profit_r`
- `max_holding_bars`: 从 `backtest.yaml` 的 `rr.max_holding_bars` 或 `labels.yaml` 的 `generator.params.max_holding_bars`

### 使用优化参数

如果提供了 `--rule-params` 参数，会覆盖配置中的值：
```bash
mlbot diagnose model-comparison \
  --strategy-config config/strategies/sr_reversal_long \
  --rule-params results/optimization/best_params.csv \
  ...
```

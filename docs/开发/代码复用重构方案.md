# 代码复用重构方案：model-comparison 与 strategy-feature-compare

## 📊 问题描述

`mlbot diagnose model-comparison` 和 `mlbot analyze strategy-feature-compare` 存在大量代码重复：

### 当前情况

| 功能 | model-comparison | strategy-feature-compare |
|------|------------------|--------------------------|
| **特征加载** | 自己实现（直接调用 `feature_loader.load_features_from_requested`） | 使用 `strategy_runner.run_feature_pipeline` |
| **标签生成** | 硬编码 `compute_rr_label` | 从配置导入标签生成函数 |
| **模型训练** | 硬编码 `train_ml_model` | 从配置导入训练函数 |
| **回测** | 简化评估（`evaluate_ml_model`） | 完整回测（`strategy_runner.run_vectorbt_backtest`） |

### 问题影响

1. ⚠️ **代码重复**：相同的逻辑在两个地方实现
2. ⚠️ **维护困难**：修改需要同步两个地方
3. ⚠️ **行为不一致**：可能导致结果差异
4. ⚠️ **配置不统一**：`model-comparison` 硬编码参数，`strategy-feature-compare` 使用配置

---

## ✅ 重构方案

### 方案A：让 model-comparison 复用 strategy-feature-compare 的代码（推荐）

**核心思想**：让 `model-comparison` 使用 `train_strategy_pipeline.py` 中的函数

#### 步骤1：提取公共函数到 `train_strategy_pipeline.py`

```python
# 在 train_strategy_pipeline.py 中添加公共函数

def load_features_and_labels(
    df_train: pd.DataFrame,
    df_test: pd.DataFrame,
    strategy_config: StrategyConfig,
    feature_loader: StrategyFeatureLoader,
) -> Tuple[pd.DataFrame, pd.DataFrame, List[str]]:
    """
    加载特征和标签（公共函数）
    
    返回：
        df_train_features: 训练集特征
        df_test_features: 测试集特征
        feature_cols: 特征列名列表
    """
    # 1. 加载特征
    df_train_features = run_feature_pipeline(
        df_train, feature_loader, strategy_config.features, fit=True
    )
    df_test_features = run_feature_pipeline(
        df_test, feature_loader, strategy_config.features, fit=False
    )
    
    # 2. 生成标签
    label_func = import_callable(
        strategy_config.labels.generator.module,
        strategy_config.labels.generator.function,
    )
    target_col = strategy_config.labels.target_column
    df_train_features[target_col] = label_func(
        df_train_features.copy(), **strategy_config.labels.generator.params
    )
    df_test_features[target_col] = label_func(
        df_test_features.copy(), **strategy_config.labels.generator.params
    )
    
    # 3. 应用过滤器
    df_train_filtered = apply_filters(
        df_train_features, strategy_config.labels.filters
    )
    df_test_filtered = apply_filters(
        df_test_features, strategy_config.labels.filters
    )
    
    # 4. 确定特征列
    feature_cols = determine_feature_columns(
        df_train_filtered, strategy_config.features
    )
    
    return df_train_filtered, df_test_filtered, feature_cols


def train_model_from_config(
    df_train: pd.DataFrame,
    df_test: pd.DataFrame,
    strategy_config: StrategyConfig,
    feature_cols: List[str],
) -> Tuple[Any, Dict[str, float], Any]:
    """
    从配置训练模型（公共函数）
    
    返回：
        models: 训练好的模型
        metrics: 评估指标
        preprocessor: 预处理器
    """
    trainer_func = import_callable(
        strategy_config.model.trainer.module,
        strategy_config.model.trainer.function,
    )
    trainer_params = dict(strategy_config.model.trainer.params)
    target_col = trainer_params.pop(
        "target_col", strategy_config.labels.target_column
    )
    
    models, avg_metric, cv_results, used_features, preprocessor = trainer_func(
        df_train,
        feature_cols=feature_cols,
        target_col=target_col,
        **trainer_params,
    )
    
    metrics = {
        "avg_cv_metric": avg_metric,
        "cv_results": cv_results,
    }
    
    return models, metrics, preprocessor
```

### 方案B：创建独立的公共模块（更彻底）

**核心思想**：创建 `src/time_series_model/pipeline/common/` 模块，提取所有公共函数

#### 目录结构

```
src/time_series_model/pipeline/common/
├── __init__.py
├── feature_loading.py      # 特征加载公共函数
├── label_generation.py      # 标签生成公共函数
├── model_training.py       # 模型训练公共函数
└── backtesting.py          # 回测公共函数
```

#### 示例：`feature_loading.py`

```python
"""公共特征加载函数"""

from typing import List, Tuple
import pandas as pd
from src.features.loader.strategy_feature_loader import StrategyFeatureLoader
from scripts.train_strategy_pipeline import run_feature_pipeline

def load_strategy_features(
    df_train: pd.DataFrame,
    df_test: pd.DataFrame,
    features_config,
    feature_loader: StrategyFeatureLoader,
) -> Tuple[pd.DataFrame, pd.DataFrame, List[str]]:
    """
    加载策略特征（公共函数）
    
    Args:
        df_train: 训练集原始数据
        df_test: 测试集原始数据
        features_config: 特征配置
        feature_loader: 特征加载器
    
    Returns:
        df_train_features: 训练集特征
        df_test_features: 测试集特征
        feature_cols: 特征列名列表
    """
    # 加载特征
    df_train_features = run_feature_pipeline(
        df_train, feature_loader, features_config, fit=True
    )
    df_test_features = run_feature_pipeline(
        df_test, feature_loader, features_config, fit=False
    )
    
    # 确定特征列
    from scripts.train_strategy_pipeline import determine_feature_columns
    feature_cols = determine_feature_columns(df_train_features, features_config)
    
    return df_train_features, df_test_features, feature_cols
```

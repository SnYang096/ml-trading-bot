**归一化状态**: ⚠️ **部分归一化**

### 问题分析

#### 3.1 使用 rank() 和 scale() 函数
```python
# alpha_utils.py
def rank(df):  # 返回 0-1 之间的分位数 ✅
    return df.rank(axis=1, pct=True)

def scale(df, k=1.0):  # 归一化到 [-k, k] ✅
    return df.div(df.abs().sum(axis=1)).mul(k)
```

#### 3.2 但原始 alpha101_raw 可能包含未归一化的值

#### 修复 2: Alpha101 特征值范围检查
在 `comprehensive_feature_engineering.py` 中，Alpha101 特征工程后检查并归一化：

```python
# 3. Alpha101 因子特征
if self.use_alpha101:
    alpha_df = self.alpha101_engineer.compute(alpha_source, required_features)
    df = df.join(alpha_df, how="left")
    
    # 🔧 添加：检查并归一化大值特征
    for col in alpha_df.columns:
        if col in df.columns:
            col_values = df[col].dropna()
            if len(col_values) > 0:
                col_abs_max = col_values.abs().max()
                if col_abs_max > 1000:  # 值范围过大，需要归一化
                    from sklearn.preprocessing import RobustScaler
                    scaler = RobustScaler()
                    df[col] = scaler.fit_transform(df[[col]].values).flatten()
```

## 验证方法

运行训练时，检查特征分布：

```python
# 检查特征分布是否相似（如果归一化正确，均值应该相似）
for symbol in train_df["symbol"].unique():
    symbol_X = X_df[train_df["symbol"] == symbol]
    feature_means = symbol_X.mean()
    feature_stds = symbol_X.std()
    
    # 如果均值或标准差过大（>1000），可能是未归一化的特征
    if feature_means.abs().max() > 1000 or feature_stds.max() > 1000:
        print(f"⚠️ {symbol} 有未归一化的特征")
```

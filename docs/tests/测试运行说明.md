# 测试运行说明

## 运行所有特征测试

### 方法 1: 使用 Makefile（推荐）

```bash
# 运行所有关键特征测试
make test-key-features-all

# 运行所有复杂特征测试
make test-complex-features-comprehensive

# 运行所有特征的综合测试
make test-all-features-comprehensive
```

### 方法 2: 使用脚本

```bash
# 运行所有测试
bash scripts/run_all_feature_tests.sh
```

### 方法 3: 单独运行

```bash
# VPIN 测试
make test-vpin-future-leak

# WPT 测试
make test-wpt-future-leak

# Volume Profile Volatility 测试
make test-volume-profile-volatility-future-leak

# 复杂特征测试
make test-complex-features-comprehensive
```

## 注意事项

1. **需要 Docker**: 所有测试都在 Docker 容器中运行
2. **依赖包**: 测试会自动安装所需的包（pytest, scipy, arch 等）
3. **运行时间**: 完整测试可能需要几分钟时间

## 故障排除

如果测试失败：
1. 检查 Docker 是否运行: `docker ps`
2. 查看详细日志: 测试会输出详细的错误信息
3. 检查依赖: 确保所有必需的 Python 包都已安装

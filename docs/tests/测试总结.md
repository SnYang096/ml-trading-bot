## 修复的问题

### 1. ATR 计算数组长度不匹配

**问题**：`_update_timeframe_features` 中 ATR 计算时数组长度不匹配。

**修复**：添加数组长度检查和调整逻辑。

### 2. pytest asyncio marker

**问题**：pytest 配置中缺少 `asyncio` marker。

**修复**：在 `pytest.ini` 中添加 `asyncio` marker。

### 3. Nautilus TradeTick 创建

**问题**：TradeTick 需要使用正确的类型（Price、Quantity 对象）。

**修复**：使用 `Price.from_str()` 和 `Quantity.from_str()` 创建对象。

### 4. 异步测试简化

**问题**：WebSocket 异步测试过于复杂。

**修复**：简化测试，主要测试逻辑结构，不测试实际 WebSocket 连接。

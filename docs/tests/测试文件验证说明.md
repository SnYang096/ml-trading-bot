## ⚠️ 可能的问题和解决方案

### 1. 缺少依赖包

**问题**: `ModuleNotFoundError: No module named 'arch'` 或 `scipy`

**解决方案**:
```bash
pip install arch scipy
```

**注意**: 如果包不可用，GARCH 和 EVT 特征会返回空特征，但不会报错。

---

### 2. 测试断言失败

**可能原因**:
- 随机数种子不同导致结果差异
- 模拟数据不够稳定

**解决方案**:
- 检查测试中的 `np.random.seed(42)` 是否一致
- 调整断言容差（delta 参数）

---

### 3. 测试运行缓慢

**原因**: 
- GARCH 拟合需要时间
- 大量数据计算

**解决方案**:
- 减少测试数据量（`n_samples`）
- 使用 `update_freq > 1` 提升效率

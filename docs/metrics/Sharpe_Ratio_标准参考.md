# Sharpe Ratio 标准参考

## 📊 Sharpe Ratio 定义

**Sharpe Ratio = (平均收益 - 无风险利率) / 收益标准差**

衡量**风险调整后的收益**，即每承担一单位风险能获得多少超额收益。

## 🎯 一般标准

### 传统金融市场

| Sharpe Ratio | 评级 | 说明 |
|-------------|------|------|
| **< 0** | 很差 | 收益低于无风险利率 |
| **0 - 0.5** | 较差 | 勉强可接受，但风险调整后收益低 |
| **0.5 - 1.0** | 一般 | 可接受的水平 |
| **1.0 - 2.0** | 良好 | 不错的风险调整后收益 |
| **2.0 - 3.0** | 优秀 | 非常好的表现 |
| **> 3.0** | 卓越 | 极少数策略能达到 |

### 量化交易标准

| Sharpe Ratio | 评级 | 说明 |
|-------------|------|------|
| **< 0.5** | 不合格 | 不建议实盘使用 |
| **0.5 - 1.0** | 可接受 | 可以小仓位测试 |
| **1.0 - 1.5** | 良好 | 可以考虑实盘 |
| **1.5 - 2.0** | 优秀 | 实盘推荐 |
| **2.0 - 3.0** | 卓越 | 顶级策略 |
| **> 3.0** | 可疑 | 可能存在数据泄漏或过拟合 |

## 📈 不同市场的合理范围

### 股票市场（长期）

- **被动投资（指数基金）**: 0.5 - 1.0
- **主动管理基金**: 0.5 - 1.5
- **对冲基金**: 1.0 - 2.0
- **量化策略**: 1.0 - 2.5

### 期货市场

- **CTA 策略**: 0.8 - 1.5
- **高频交易**: 2.0 - 5.0（但样本可能不稳定）
- **中低频策略**: 1.0 - 2.0

### 加密货币市场

- **波动性大，Sharpe Ratio 通常较低**
- **合理范围**: 0.5 - 1.5
- **> 2.0 可能有问题**（市场效率低，但风险极高）

## ⚠️ 重要注意事项

### 1. 年化 vs 非年化

**当前你的计算**：
- 交易级别的 Sharpe Ratio = 0.282
- 这是**非年化**的

**如果年化**（假设每年 50 个交易）：
- 年化 Sharpe Ratio ≈ 0.282 * sqrt(50) ≈ **2.0**

**结论**：你的 0.282 如果年化后可能达到 1.5-2.0，这是**良好的水平**！

### 2. 样本数量影响

- **> 100 个交易**：非常可靠

- **分钟线策略**：Sharpe Ratio 可能 1.0 - 3.0（但需要更多样本）
- **高频策略**：Sharpe Ratio 可能 2.0 - 5.0（但样本要求极高）

### 4. 市场环境影响

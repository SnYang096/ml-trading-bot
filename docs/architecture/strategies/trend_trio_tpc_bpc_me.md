# 趋势三兄弟：TPC / BPC / ME 语义对比

> 三个策略都是抓趋势，但抓的是**不同阶段**或**不同形态**的趋势机会。

---

## 一、核心语义一句话

| 策略 | 一句话 | 市场阶段 |
|------|--------|----------|
| **ME** | 动量爆炸，直接入场 | 趋势**启动 / 加速**阶段 |
| **BPC** | 突破→回调→延续确认后入场 | 趋势**结构确认后的续涨**阶段 |
| **TPC** | 趋势中回调不破，吸收后顺势入场 | 趋势**回撤修正**阶段 |

---

## 二、语义流程对比

```
价格时间轴 ────────────────────────────────────────────────►

    ┌─ ME: 这里动量爆发，不等回调，直接进
    │
    │       ┌─ BPC: 突破发生了，等它回调，回调后恢复 → 进
    │       │
    │       │           ┌─ TPC: 大趋势中出现回调，订单流说逆向力量衰竭 → 进
    │       │           │
────╱╲──╱╲╲╱──────╱╲──╱╲╲╱──────╱╲──╱╲╲╱──────────────
   爆发   回调  延续   爆发  回调  延续   回调  吸收  顺势
```

## 三、详细对比

### 3.1 Prefilter（什么场景才值得看）

| 维度 | ME | BPC | TPC |
|------|-----|-----|-----|
| 波动环境 | `me_atr_pct >= 0.25`（极松，排除死盘） | `bpc_volume_compression_pct >= 0.89` | `bpc_volume_compression_pct >= 0.89` |
| 动量/回调 | `me_accel_persistence >= 0.4`（连续加速） | `bpc_pullback_depth <= 0.55`（回调不过深） | `tpc_pullback_depth <= 0.75`（更宽松，只要不破结构） |
| 方向一致性 | `me_multi_tf_alignment >= 0.5`（多 TF 共振） | `bpc_recovery_strength >= 0.6`（恢复强度） | 无（由 entry filter 控制） |
| 是否需要突破 | **不需要** | **需要**（breakout-gated） | **不需要** |

### 3.2 Entry Filter（什么信号触发入场）

| 维度 | ME | BPC | TPC |
|------|-----|-----|-----|
| 核心逻辑 | 订单流共识 + 拥挤度 + VPIN | 突破后延续的订单流 | 吸收 + 恢复顺势 |
| 关键特征 | `me_cvd_strength`, `ofci_pct`, `shd_pct`, `vpin_zscore` | 14 条候选（过于复杂）| `tpc_cvd_absorption >= 1.0` + `tpc_recovery_strength >= 0.15` |
| 组合模式 | OR（任一成立即入场） | OR（但全部 disabled） | OR（两条对称：做多/做空） |
| 简洁度 | 中等（5 条 OR） | **极低**（14 条，均 disabled） | **极高**（2 条，清晰语义） |

### 3.3 Gate（什么环境必须回避）

三个策略共享同一套硬门控框架：

| 共享 Gate | ME | BPC | TPC |
|-----------|:--:|:---:|:---:|
| semantic_chop（震荡区） | > 0.62 | > 0.90（放宽） | > 0.62 |
| EMA1200 追涨/杀跌 | > 0.10 / < -0.10 | > 0.10 / < -0.10 | > 0.10 / < -0.10 |
| EVT 尾部风险 | — | ✓ | ✓ |
| VPIN 知情交易 | — | ✓ | ✓ |
| VWAP 近场死区 | — | ✓ | ✓ |
| vol_persistence | — | ✓ | ✓ |

## 四、关键区别总结

| | ME | BPC | TPC |
|---|---|---|---|
| **抓什么** | 动量爆发的初始推力 | 突破→回调→延续的完整结构 | 趋势内回调的吸收拐点 |
| **不需要什么** | 不需要回调 | 不需要（已有突破先验） | 不需要突破 |
| **入场依据** | 资金集中涌入 | 回调恢复 + 订单流延续 | 逆向力量衰竭（吸收语义） |
| **风险** | 假突破 / 尾部追涨 | 假延续 / 回调过深 | 回调不止 / 趋势反转 |
| **时间偏好** | 最快（抓爆发点） | 中等（等结构完成） | 较慢（等吸收确认） |

---

## 五、互补关系

三个策略在趋势的不同阶段接力，理论上互补：

1. **ME** 最先入场 → 捕捉爆发段利润
2. 爆发后价格回调 → **BPC** 在结构确认后的延续入场
3. 大趋势中反复回调修正 → **TPC** 在每次吸收后加仓或重新入场

同一段趋势中，ME 可能只触发一次，BPC 在结构突破后触发，TPC 可以反复触发（每次回调都是机会）。

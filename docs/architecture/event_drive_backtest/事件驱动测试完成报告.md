# 事件驱动相关测试说明

本报告描述 **`tests/event_driven/`** 下与「增量特征 + WebSocket 客户端」相关的自动化测试，与 **`scripts/event_backtest.py`** 的单元测试（`tests/unit/test_event_*.py` 等）是不同目录；后者覆盖时间对齐、macro VWAP、resume 等业务逻辑。

## 测试范围（`tests/event_driven/`）

| 文件 | 内容 |
|------|------|
| `test_incremental_feature_computer.py` | `IncrementalFeatureComputer`：tick/bar、VPIN、订单流、缓冲区等 |
| `test_websocket_client.py` | `BinanceWebSocketClient`：URL、回调、错误处理等（不连真实网络为主） |
| `test_integration.py` | tick→特征、多周期、回调链等集成向用例 |

**说明**：历史上部分草稿曾列出 `test_event_driven_strategy.py`；**当前仓库该文件不存在**，请勿在 CI 或文档中引用。

## 运行

```bash
python -m pytest tests/event_driven/ -v
python -m pytest tests/event_driven/ --collect-only -q
```

最近一次在本仓库 **`pytest tests/event_driven/ --collect-only`** 收集到 **33** 条用例；**是否全部通过取决于环境与依赖**，请以你本机 `pytest tests/event_driven/ -v` 结果为准（勿依赖历史报告中的「全绿」快照）。

## 覆盖率（可选）

```bash
python -m pytest tests/event_driven/ \
  --cov=src.time_series_model.live \
  --cov=src.live_data_stream \
  --cov-report=term-missing
```

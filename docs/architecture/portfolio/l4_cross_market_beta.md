# L4：跨市场 β 轮动（组合层 — 与「漏斗 L4 Gate」无关）

## 命名说明

- **实施文档 04**（`docs/z实验_005_统一研究/实施文档_04_整体投资架构_alpha_beta_跨市场_三层仓位.md`）中，**加密内部**只有 **L1 现货 / L2 合约 / L3 彩票**。
- **加密之外**，同一文档 **§2** 描述了 **美股 ETF / 黄金 / 长债** 等 **跨市场 β**，作为降低单一加密市场风险、与合约 α 解耦的配置层。
- 在本仓库里，把这一层单独记为 **组合层 L4**，仅表示 **「传统资产/跨市场动量」**，**不是** `GenericLiveStrategy` 里 Prefilter/Direction/Gate/**Evidence** 那套「第四层漏斗」。

## L4 在整体结构中的位置

```text
         ┌─────────────────────────────────────────┐
         │  L4 跨市场 β（SPY/QQQ/GLD/TLT 月度动量等）  │
         └──────────────────┬──────────────────────┘
                            │
         ┌──────────────────▼──────────────────────┐
         │  加密侧：L1 现货 β / L2 合约 α / L3 彩票   │
         └──────────────────────────────────────────┘
```

## 操作化来源

- **策略语义**：月度再平衡、12 个月绝对收益（与 12M−1M 动量可作为补充），持有前 **1～2 名**，见实施文档 §2.3。
- **工程配置**：`config/portfolio/l4_cross_market_momentum.yaml`
- **离线计算**：`scripts/l4_cross_market_momentum.py`（读月度收盘价 CSV + 输出目标权重摘要）

数据来源需自行合规取得（券商导出、指数公司、或受许可的行情 API）；脚本不内置抓数，避免环境差异与协议问题。

## 与 L3 彩票的关系

- **正交**：L4 解决「钱放在哪类资产」；L3 解决「加密内用利润博极端倍数」。
- **资金路径**：典型做法是 L2 利润 → 按家族预算划到 L3；**全组合**层面的 L4 再平衡可与 L1/L2 **不同账户**执行，避免混合保证金与心理账户。

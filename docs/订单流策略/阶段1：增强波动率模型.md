# 1. 目标变量：未来5分钟已实现波动率（RV）
df['rv_future_5'] = df['returns'].rolling(window=5).std().shift(-5) # 注意：实际用 future window

# 2. VPIN volatility 特征
df['vpin_vol_10'] = df['vpin'].rolling(10).std()
df['vpin_vol_20'] = df['vpin'].rolling(20).std()

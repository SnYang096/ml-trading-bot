## 🚀 二、如何设计“动态仓位管理”？

### ✅ 1. **基础：波动率自适应仓位**（必须做）

```python
def calculate_position_size(self):
    # 基于 ATR 的风险单位
    atr = self.atr[0]
    price = self.data.close[0]
    risk_per_share = atr * self.p.stop_atr_mult  # 每单位亏损
    account_risk = self.broker.getvalue() * self.p.risk_per_trade_pct  # 单笔最大亏损
    
    size = account_risk / risk_per_share
    return max(size, self.min_size)
```

👉 这是底线，确保**每笔交易风险一致**。

---

### ✅ 2. **趋势强度加仓**（解决“趋势中仓位不足”）

#### 思路：
- 用 LightGBM 的**预测概率**作为“趋势置信度”
- 概率越高，仓位越大

```python
class ProbabilisticSizer:
    def get_size(self, base_size, ml_prob):
        # ml_prob: P(up) from LightGBM
        if ml_prob < 0.5:
            return 0
        elif ml_prob < 0.6:
            return base_size * 0.5
        elif ml_prob < 0.7:
            return base_size * 1.0
        elif ml_prob < 0.8:
            return base_size * 1.5
        else:
            return base_size * 2.0  # 强趋势，重仓
```

> ✅ 这样你就能在“强趋势”中自动提高仓位。

---

### ✅ 3. **金字塔加仓**（Pyramiding）

#### 规则：
- 只在盈利后加仓
- 每次加仓仓位递减
- 加仓点必须满足原信号条件

```python
def next(self):
    if self.position.size == 0:
        # 初始开仓
        size = self.sizer.get_size(ml_prob=self.ml_score)
        self.buy(size=size)
    elif self.position.profit > 0:
        # 盈利后，检查是否满足加仓条件
        if self.is_reentry_condition() and self.ml_score > 0.65:
            add_size = self.sizer.get_size(ml_prob=self.ml_score) * 0.5  # 递减
            self.buy(size=add_size)
```

> ✅ 避免“倒金字塔”（亏损加仓），只在趋势确认后加码。

## 🛑 三、智能止损与止盈设计

### ✅ 1. **动态移动止损**（Trailing Stop）

不要用固定 ATR 倍数，而是：

```python
def next(self):
    if self.position.size > 0:
        # 基于小波趋势的移动止损
        price_trend = self.wavelet_price_trend  # 低频成分
        new_sl = price_trend * 0.98  # 略低于趋势线
        self._place_or_move_stop(new_sl)
```

👉 这样在强趋势中，止损会“贴”着趋势走，避免被震荡洗出。

---

### ✅ 2. **分级止盈**（Take-Profit Laddering）

```python
# TP1: 1x R → 50% 平仓
# TP2: 2x R → 30% 平仓
# TP3: 3x R → 20% 平仓
```

```python
if profit >= 1.0 * R and not self.tp1_done:
    self.sell(size=self.position.size * 0.5)
    self.tp1_done = True

if profit >= 2.0 * R and not self.tp2_done:
    self.sell(size=self.position.size * 0.6)  # 剩余的 30%
    self.tp2_done = True
```

> ✅ 让利润奔跑，同时锁定部分收益。

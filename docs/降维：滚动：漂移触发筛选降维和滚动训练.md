def calculate_psi(expected, actual, bins=10):
# 计算 Population Stability Index
cutoffs = np.histogram_bin_edges(expected, bins)
expected_p = np.histogram(expected, bins=cutoffs)[0] / len(expected)
actual_p = np.histogram(actual, bins=cutoffs)[0] / len(actual)
psi = np.sum((expected_p - actual_p) * np.log((expected_p + 1e-8) / (actual_p + 1e-8)))
return psi

📌 五、关于你脚本中 Autoencoder 的使用建议

你用 Autoencoder 做降维是合理的，但要注意：

优点 缺点
------ ------
✅ 无监督，可保留非线性结构 ❌ 压缩后的特征无可解释性
✅ 适合高维连续特征 ❌ 不能直接用于特征选择（你不知道哪个原始特征重要）
✅ 可与其他方法结合 ❌ 训练成本高

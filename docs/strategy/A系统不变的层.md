# A 系统不变的层

A 系统比 B 系统简单很多，需要调整的地方也少很多。

---

## 核心假设

**周线 EMA200 以下 = 深熊 = 值得积累。**

该假设在 crypto 历史上较稳定，不需频繁重验，只需确认仍有效。

---

## 真正需要调整的两处

### 卖出：5× 阈值

唯一宜**人工判断**的参数：周期牛市高度不同，5× 可能偏保守或偏激进。属周期判断，不必自动化。

### 币种选择

当前 BTC / BNB / SOL（及少量 alt 配置）。组合定好后整周期少动；仅当出现根本性破坏（监管、基本面）才调整。

---

## 不需要调整的地方

| 项 | 说明 |
| --- | --- |
| 买入 | 周线 EMA200 以下定投，极简 |
| Deploy decay | 越买越慢，敏感度低 |
| 卖出节奏 | 幂律加速已编码「慢卖、越涨越快」 |
| 止损 | 无止损正确；退出语义是周期非波动 |

---

## 唯一运维工作

**定期确认：价是否仍在周线 EMA200 以下？**

- 在 → 继续买  
- 不在 → 停买，等卖出条件  

其余交给规则执行。

---

## 一句话

**A 已足够简单。唯一人工判断是周期卖出目标倍数；其余全自动。**

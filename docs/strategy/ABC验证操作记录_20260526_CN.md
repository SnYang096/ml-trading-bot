### quick_layer_scan（3 模式 + bucket-by）

```bash
PARQ=results/validation_smoke/tpc/features_labeled.parquet
OUT=results/validation_smoke/tpc

PYTHONPATH=src:scripts python scripts/quick_layer_scan.py feature-plateau \
  --features-parquet "$PARQ" --label success_no_rr_extreme \
  --feature tpc_pullback_depth --operator "<=" \
  --grid 0.5,0.6,0.7,0.75,0.8 --filter "tpc_semantic_chop<=0.4" \
  --out "$OUT/quick_scan/pullback_plateau.md"

PYTHONPATH=src:scripts python scripts/quick_layer_scan.py condition-set \
  --features-parquet "$PARQ" --label success_no_rr_extreme \
  --filter "tpc_semantic_chop<=0.4" \
  --condition "H: abs(ema_1200_position)>0.10" \
  --condition "F: abs(ema_1200_position)>0.12" \
  --out "$OUT/quick_scan/regime_candidates.md"

PYTHONPATH=src:scripts python scripts/quick_layer_scan.py ic-decay \
  --features-parquet "$PARQ" \
  --features ema_1200_position,vol_persistence,tpc_pullback_depth \
  --horizons 1,3,5,10,20 \
  --out "$OUT/quick_scan/ic_decay.md"

PYTHONPATH=src:scripts python scripts/quick_layer_scan.py condition-set \
  --features-parquet "$PARQ" --label success_no_rr_extreme \
  --filter "tpc_semantic_chop<=0.4" \
  --condition "H: abs(ema_1200_position)>0.10" \
  --bucket-by "ema:ema_1200_position@0.10" \
  --out "$OUT/quick_scan/regime_bucketed.md"
```

| 报告 | 判读（合成数据） | ✅ |
|---|---|---|
| `regime_candidates.md` | H: Δpp **+10.75pp**, \|z\|=17.42 | 链路 OK；数值无实盘意义 |
| `regime_bucketed.md` | 两桶各出 condition 表 | bucket-by OK |
| `pullback_plateau.md` | plateau 表生成 | OK |

一、GARCH/FIGARCH：波动聚集性 & 杠杆效应
🔍 1. 什么是波动聚集性（Volatility Clustering）？
现象：金融市场中，高波动期往往接高波动，低波动期接低波动——波动不是随机的，而是“成群出现”。

✅ 案例：
2020年3月美股熔断后，连续多日ATR飙升 → 高波动持续
2023年BTC横盘3个月，ATR极低 → 低波动持续

但滚动标准差或ATR只是描述过去，无法预测未来波动是否持续。
📈 2. GARCH(1,1) 模型如何捕捉它？
模型公式：
\[
\sigma_t^2 = \omega + \alpha r_{t-1}^2 + \beta \sigma_{t-1}^2
\]
\(\omega\)：长期平均波动（基线）
\(\alpha\)：对昨日冲击的反应（新闻/事件影响）
\(\beta\)：对历史波动的记忆（持续性）

👉 关键指标：Persistence = \(\alpha + \beta\)
若 \(\alpha + \beta \approx 0.95\) → 波动会持续很久（如危机后）
若 \(\alpha + \beta \approx 0.7\) → 波动快速衰减（正常市场）
⚖️ 3. 杠杆效应（Leverage Effect）是什么？
现象：下跌时波动率上升更快（因为公司杠杆增加，风险上升），上涨时波动上升较慢。

✅ GJR-GARCH 或 EGARCH 可建模此不对称性：
负收益对 \(\sigma_t^2\) 的贡献 > 正收益（同幅度）
💡 在你的策略中怎么用？

策略 应用方式
------ --------
压缩区突破策略 当前 ATR 低 + GARCH persistence 高 → 表明“低波动是暂时的”，突破概率大，可提高仓位
趋势策略 GARCH volatility 上升 + trend_r2_20 > 0.6 → 趋势+高波动=强趋势，加仓；若 volatility 上升但 trend_r2 低 → 假突破，观望
SR反转策略 在 SR 附近，若 GARCH volatility 突增但 price 未破位 → 可能是“恐慌抛售”，适合反手做多
✅ 输出特征直接喂给模型：garch_volatility, garch_persistence 是连续值，树模型可自动学习阈值。

二、VPIN：为什么比 taker_buy_ratio 更好？
🧠 1. taker_buy_ratio 的缺陷
python
tbr = buy_volume / total_volume
问题1：受时间窗口影响大（1分钟 vs 1小时结果不同）
问题2：无法区分“100笔小单买入”和“1笔大单买入”——后者更可能含信息
🔬 2. VPIN 的核心思想：按成交量分桶（Volume Clock）
Easley & O’Hara (2012) 提出：交易信息流应以“成交量”而非“时间”为单位。
步骤：
1. 将总成交量切成等量“桶”（如每桶 100 BTC）
2. 在每个桶内计算：
\[
\text{VPIN} = \frac{ V_{\text{buy}} - V_{\text{sell}} }{V_{\text{buy}} + V_{\text{sell}}}
\]
3. VPIN 越高 → 买卖不平衡越严重 → 知情交易者可能在行动
✅ 你有 tick 数据，可以做得更精细！
用 逐笔成交方向（非 OHLC 推断）判断 buy/sell
桶大小可动态调整（如 0.1% 日均成交量）
💡 策略应用

策略 应用方式
------ --------
SR突破策略 突破瞬间 VPIN > 0.6 → 真突破；VPIN < 0.3 → 假突破（无大单跟随）
压缩区突破 压缩末期 VPIN 持续上升 → 资金在悄悄建仓，预示即将爆发
趋势策略 趋势中 VPIN 与价格同向 → 健康趋势；VPIN 与价格背离 → 趋势衰竭
📌 优势：VPIN 对“大单冲击”敏感，而 taker_buy_ratio 会被刷单/高频噪音淹没。

四、EVT 极值理论：黑天鹅预警
🌪 1. 传统方法的不足
滚动 VaR 假设正态分布 → 低估尾部风险
ATR 百分位只能告诉你“当前是否极端”，不能预测“是否会更极端”
📐 2. POT + GPD 如何工作？
设定一个高阈值 \(u\)（如 95% 分位数）
只关注超过 \(u\) 的“超额损失” \(y = x - u\)
用 广义帕累托分布（GPD） 拟合这些超额值：
\[
G(y) = 1 - \left(1 + \xi \frac{y}{\sigma}\right)^{-1/\xi}
\]
关键参数 \(\xi\)（shape parameter）：
\(\xi > 0\)：厚尾（无上界，如股市崩盘）
\(\xi = 0\)：指数尾（如 Poisson 过程）
\(\xi < 0\)：短尾（有最大值，如保险赔付）
📉 3. \(\xi\) 突增意味着什么？
正常市场：\(\xi \approx 0.1 - 0.3\)
危机前兆：\(\xi\) 从 0.2 → 0.6 → 表明极端事件概率急剧上升
💡 策略应用

策略 应用方式
------ --------
所有策略 当 \(\xi > 0.5\) 且 rising → 降低仓位，因黑天鹅风险高
趋势策略 高 \(\xi\) + 趋势 → 可能是“疯牛/疯熊”，但随时反转，需收紧止损
压缩突破 压缩末期 \(\xi\) 下降 → 市场稳定，突破更可靠
✅ 可直接放入模型：evt_tail_shape 是连续特征，树模型会自动学习“\(\xi > 0.5\) 时减少多头敞口”。

✅ 你的说法是对的，**时序模型也可以使用横截面因子**，但需要注意方式和语义。下面详细解释：

---

### 1️⃣ **时序模型 vs 横截面因子**

* **时序模型**关注一个资产随时间变化的动态，例如预测 BTC 下一小时收益
* **横截面因子**通常是同一时刻多个资产之间的关系，例如 BTC 与 ETH 的价差、相对强弱、市场普涨/普跌信号

> 本质上，横截面因子可以作为 **时序特征** 的一部分，只要你能在时间轴上对齐数据。

---

### 2️⃣ **使用方式**

1. **单资产时序模型 + 横截面特征**

   * 对每个时刻，把其他币种的信号、涨跌幅或指标作为额外特征输入
   * 示例：

     ```text
     t = now
     X_BTC_t = [BTC_rsi, BTC_atr, ETH_return, SOL_return, ...]
     y_BTC_t = BTC_future_return_1h
     ```
   * 模型学到的就是 BTC 的时序行为 + 横截面信息对未来收益的影响

2. **多资产统一模型**

   * 将多个币种的特征一起输入统一模型，训练一个多资产预测器
   * 特征可以包含：

     * 本币历史技术指标（时序特征）
     * 其他币种当期指标/涨跌（横截面特征）
     * Regime 或状态特征

3. **共振逻辑**

   * 横截面因子可以帮助捕捉**市场整体风格**，例如“全币普涨 → 牛市趋势加强”
   * 对仓位、杠杆、止损策略可直接提供信号

---

### 3️⃣ **注意事项**

* **时间对齐**：确保横截面特征在同一时间戳，否则会引入未来函数
* **信息泄露**：多币种特征必须严格使用过去时刻的数据
* **特征规模差异**：不同币波动率不同，可做标准化/归一化
* **特征膨胀**：币种越多，横截面特征增长快，注意防止过拟合

---

### 4️⃣ **总结**

* 时序模型不仅可以，也**应当**使用横截面因子，特别是在多币种、市场风格波动明显的场景
* 横截面特征本质上是时序特征的“跨资产扩展”
* 对统一模型、Regime-aware、多周期特征体系都能直接受益

---

import pandas as pd
import numpy as np
from typing import Dict, Optional
from baseline_feature_engineering import BaselineFeatureEngineer

def make_features_labels_with_cs(
df_dict: Dict[str, pd.DataFrame],
symbol: str,
timeframe: str,
forward_bars: int,
feature_engineer: Optional[BaselineFeatureEngineer] = None,
market_cap_dict: Optional[Dict[str, pd.Series]] = None,
universe_top_n: int = 50
):
"""
为单个币种生成特征 + 横截面因子 + 标签

Args:
df_dict: {symbol: DataFrame}，原始 OHLCV 数据
symbol: 目标币种，如 'BTC'
timeframe: 该币种最优时间尺度，如 '1H'
forward_bars: 预测未来多少根K线
feature_engineer: 你的 BaselineFeatureEngineer 实例
market_cap_dict: 可选，{symbol: Series} 市值数据，用于 cap-weighted 因子
universe_top_n: 仅在 Top N 币中计算横截面（防噪声）

Returns:
pd.DataFrame: 包含 features + label + meta columns
"""
if feature_engineer is None:
feature_engineer = BaselineFeatureEngineer()

# Step 1: 重采样目标币数据作为时间锚点
df_target_raw = df_dict[symbol].copy()
df_target_resampled = df_target_raw.resample(timeframe).agg({
'open': 'first',
'high': 'max',
'low': 'min',
'close': 'last',
'volume': 'sum'
}).dropna()

if len(df_target_resampled) < 50:
raise ValueError(f"Not enough data for {symbol} at {timeframe}")

# Step 2: 为所有币种生成基础特征，并对齐到目标时间轴
engineered_dict = {}
valid_symbols = []

for sym, raw_df in df_dict.items():
try:
resampled = raw_df.resample(timeframe).agg({
'open': 'first',
'high': 'max',
'low': 'min',
'close': 'last',
'volume': 'sum'
}).dropna()

if len(resampled) < 30:
continue

eng = feature_engineer.engineer_features({'main': resampled})['main']
# 对齐时间戳（只保留 df_target_resampled 中的时间点）
eng_aligned = eng.reindex(df_target_resampled.index, method='ffill')
if eng_aligned.isnull().all().all():
continue

engineered_dict[sym] = eng_aligned
valid_symbols.append(sym)
except Exception as e:
print(f"Warning: Skip {sym} due to error: {e}")
continue

if not valid_symbols:
raise ValueError("No valid symbols after feature engineering")

# Step 3: 可选：限制成分股池（按平均市值或交易量）
if market_cap_dict is not None:
avg_caps = {}
for sym in valid_symbols:
if sym in market_cap_dict:
cap_series = market_cap_dict[sym].reindex(df_target_resampled.index, method='ffill')
avg_caps[sym] = cap_series.mean()
if avg_caps:
top_symbols = sorted(avg_caps.items(), key=lambda x: x[1], reverse=True)[:universe_top_n]
valid_symbols = [s for s, _ in top_symbols]
else:
# 默认按平均交易量排序
avg_vols = {
sym: engineered_dict[sym]['volume'].mean()
for sym in valid_symbols if 'volume' in engineered_dict[sym]
}
if avg_vols:
top_symbols = sorted(avg_vols.items(), key=lambda x: x[1], reverse=True)[:universe_top_n]
valid_symbols = [s for s, _ in top_symbols]

# Step 4: 构建横截面因子（在每个时间点 t 计算全市场快照）
cs_features = pd.DataFrame(index=df_target_resampled.index)
n_assets = len(valid_symbols)

# Helper: 安全获取列，缺失则返回常数
def safe_get_col(sym, col_name, default=1.0):
df_eng = engineered_dict[sym]
if col_name in df_eng.columns:
return df_eng[col_name].fillna(default)
else:
return pd.Series(default, index=df_eng.index)

# 1. 动量因子：SMA5/SMA20 ratio
mom_vals = {}
for sym in valid_symbols:
sma5 = safe_get_col(sym, 'sma_5', default=1.0)
sma20 = safe_get_col(sym, 'sma_20', default=1.0)
mom_vals[sym] = (sma5 / sma20).clip(0.5, 2.0) # 防极端值
mom_df = pd.DataFrame(mom_vals).T
cs_features['cs_mom_rank'] = mom_df.rank(axis=0, pct=True).loc[symbol]

# 2. 波动率因子：低波动更优（取负值后排名）
vol_vals = {}
for sym in valid_symbols:
vol = safe_get_col(sym, 'volatility_20', default=0.01)
vol_vals[sym] = -vol # 越低越好
vol_df = pd.DataFrame(vol_vals).T
cs_features['cs_lowvol_rank'] = vol_df.rank(axis=0, pct=True).loc[symbol]

# 3. 估值因子（简化）：价格相对近期高点位置（越低越“便宜”）
val_vals = {}
for sym in valid_symbols:
close = engineered_dict[sym]['close']
high_20 = close.rolling(20).max()
position = (close / high_20).fillna(1.0).clip(0.1, 1.0)
val_vals[sym] = position # 越小越低估
val_df = pd.DataFrame(val_vals).T
cs_features['cs_value_rank'] = val_df.rank(axis=0, pct=True).loc[symbol]

# 4. 成交量因子：相对成交量强度（volume / volume_sma_20）
vol_ratio_vals = {}
for sym in valid_symbols:
vol = engineered_dict[sym]['volume']
vol_sma = safe_get_col(sym, 'volume_sma_20', default=1.0)
ratio = (vol / vol_sma).fillna(1.0).clip(0.1, 5.0)
vol_ratio_vals[sym] = ratio
vol_ratio_df = pd.DataFrame(vol_ratio_vals).T
cs_features['cs_volume_rank'] = vol_ratio_df.rank(axis=0, pct=True).loc[symbol]

# 5. 趋势强度因子：ADX 或 MACD hist 强度
trend_vals = {}
for sym in valid_symbols:
adx = safe_get_col(sym, 'adx', default=20.0)
macd_hist = safe_get_col(sym, 'macd_hist', default=0.0)
# 综合趋势强度（可调权重）
strength = 0.6 adx + 0.4 np.abs(macd_hist) 100
trend_vals[sym] = strength
trend_df = pd.DataFrame(trend_vals).T
cs_features['cs_trend_rank'] = trend_df.rank(axis=0, pct=True).loc[symbol]

# 6. 市值调整动量（如果提供市值数据）
if market_cap_dict is not None and symbol in market_cap_dict:
cap_mom_vals = {}
for sym in valid_symbols:
if sym in market_cap_dict:
cap_series = market_cap_dict[sym].reindex(df_target_resampled.index, method='ffill')
mom_series = mom_vals[sym]
# 市值加权动量（大市值动量更可靠）
cap_mom = mom_series np.sqrt(cap_series.fillna(1e9))
cap_mom_vals[sym] = cap_mom
if cap_mom_vals:
cap_mom_df = pd.DataFrame(cap_mom_vals).T
cs_features['cs_cap_mom_rank'] = cap_mom_df.rank(axis=0, pct=True).loc[symbol]
else:
cs_features['cs_cap_mom_rank'] = cs_features['cs_mom_rank']
else:
cs_features['cs_cap_mom_rank'] = cs_features['cs_mom_rank']

# Step 5: 合并目标币自身特征 + 横截面特征
target_features = engineered_dict[symbol].copy()
final_df = pd.concat([target_features, cs_features], axis=1)

# Step 6: 生成标签（未来 log return）
future_return = (
final_df['close'].shift(-forward_bars) / final_df['close'] - 1
)
final_df['label'] = np.log1p(future_return)

# 移除未来信息（最后 forward_bars 行）
final_df = final_df.iloc[:-forward_bars].copy()

# 添加元信息
final_df['instrument'] = symbol
final_df['timeframe_group'] = timeframe

return final_df

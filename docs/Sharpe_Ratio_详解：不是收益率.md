# Sharpe Ratio 详解：不是收益率

## ❌ 常见误解

**错误理解**：
- Sharpe Ratio 1.0 = 一年收益 100%（翻倍）
- Sharpe Ratio 2.0 = 一年收益 200%（翻三倍）

**这是完全错误的！**

## ✅ 正确理解

### Sharpe Ratio 的定义

**Sharpe Ratio = (平均收益 - 无风险利率) / 收益标准差**

**关键点**：
- Sharpe Ratio 是**风险调整后的收益指标**
- 它衡量的是**每承担一单位风险能获得多少超额收益**
- **不是收益率本身**

### 实际含义

**Sharpe Ratio 1.0 意味着**：
- 平均超额收益 = 1 × 收益标准差
- 例如：如果收益标准差是 10%，那么平均超额收益是 10%

**Sharpe Ratio 2.0 意味着**：
- 平均超额收益 = 2 × 收益标准差
- 例如：如果收益标准差是 10%，那么平均超额收益是 20%

## 📊 实际例子

### 例子 1：Sharpe Ratio = 1.0

**假设**：
- 年化收益率：15%
- 无风险利率：5%
- 收益标准差：10%

**计算**：
- 超额收益 = 15% - 5% = 10%
- Sharpe Ratio = 10% / 10% = **1.0**

**结果**：
- **不是翻倍**，而是年化收益 15%
- 如果初始资金 100 万，一年后约 115 万

### 例子 2：Sharpe Ratio = 2.0

**假设**：
- 年化收益率：25%
- 无风险利率：5%
- 收益标准差：10%

**计算**：
- 超额收益 = 25% - 5% = 20%
- Sharpe Ratio = 20% / 10% = **2.0**

**结果**：
- **不是翻三倍**，而是年化收益 25%
- 如果初始资金 100 万，一年后约 125 万

### 例子 3：高波动性策略

**假设**：
- 年化收益率：50%
- 无风险利率：5%
- 收益标准差：30%

**计算**：
- 超额收益 = 50% - 5% = 45%
- Sharpe Ratio = 45% / 30% = **1.5**

**结果**：
- 虽然收益率高（50%），但 Sharpe Ratio 只有 1.5
- 因为波动性太大（30%）

## 🔍 关键理解

### 1. Sharpe Ratio 不是收益率

- **Sharpe Ratio 1.0** ≠ 一年翻倍
- **Sharpe Ratio 2.0** ≠ 一年翻三倍
- 实际收益率取决于**收益标准差**

### 2. 收益标准差很重要

**高 Sharpe Ratio + 低波动性** = 稳定收益
- Sharpe Ratio 2.0，标准差 10% → 年化收益 25%

**高 Sharpe Ratio + 高波动性** = 高收益但风险大
- Sharpe Ratio 2.0，标准差 30% → 年化收益 65%，但波动大

### 3. 复利效应

**如果持续交易多年**：
- 年化收益 25%，5 年后：100 万 → 305 万（3 倍）
- 年化收益 50%，5 年后：100 万 → 759 万（7.6 倍）

**但要注意**：
- 高收益通常伴随高波动性
- 需要承受较大的回撤风险

# 归一化模块合并说明

## 📋 合并概述

已将 `UnifiedNormalizer` 的功能整合到 `utils_normalization.py` 中，统一了归一化接口。

## ✅ 合并后的优势

1. **统一接口**：所有归一化功能都在一个模块中
2. **向后兼容**：保留所有现有函数式接口
3. **滚动归一化优先**：默认使用滚动归一化，防止未来信息泄露
4. **简化维护**：减少代码重复，统一实现

## 📦 模块位置

**统一后的模块**：`src/features/time_series/utils_normalization.py`

包含：
- 函数式接口：`normalize_series`, `normalize_by_group`, `normalize_dataframe`
- 面向对象接口：`UnifiedNormalizer` 类
- 便利函数：`create_unified_normalizer`, `normalize_features_unified`

## 🔄 迁移指南

### 从 `unified_normalization.py` 迁移

#### 旧代码（已弃用）

```python
from src.data_tools.unified_normalization import UnifiedNormalizer

normalizer = UnifiedNormalizer(
    strategy="rolling",
    scaler_type="standard",
    window_size=252,
)
normalizer.fit(train_df)
test_normalized = normalizer.transform(test_df)
```

#### 新代码（推荐）

```python
from src.features.time_series.utils_normalization import UnifiedNormalizer

normalizer = UnifiedNormalizer(
    method="zscore",  # 对应 scaler_type="standard"
    window=252,  # 对应 window_size
    group_col="_symbol",  # 多资产分组
)
normalizer.fit(train_df)
test_normalized = normalizer.transform(test_df)
```

### 主要变化

1. **导入路径**：
   - 旧：`from src.data_tools.unified_normalization import UnifiedNormalizer`
   - 新：`from src.features.time_series.utils_normalization import UnifiedNormalizer`

2. **参数名称**：
   - `strategy` → 不再需要（统一使用滚动归一化）
   - `scaler_type` → `method`（"standard" → "zscore", "minmax" → "minmax", "robust" → "robust_minmax"）
   - `window_size` → `window`

3. **默认行为**：
   - 旧：默认 `strategy="rolling"`，但 `window_size=100`
   - 新：默认 `window=252`（推荐值，防止未来信息泄露）

## ⚠️ 重要注意事项

### 1. 滚动归一化是默认推荐

```python
# ✅ 推荐：使用滚动归一化
normalizer = UnifiedNormalizer(window=252)  # 日频数据

# ❌ 不推荐：全局归一化（会触发警告）
normalizer = UnifiedNormalizer(window=None)  # 全局归一化
```

### 2. 多资产归一化

```python
# 多资产场景，按资产分组归一化
normalizer = UnifiedNormalizer(
    method="zscore",
    window=252,
    group_col="_symbol"  # 按资产分组
)
```

### 3. 特征自动分组

`UnifiedNormalizer` 会自动将特征分组为：
- `price`: 价格相关特征（SMA, EMA等）
- `ratio`: 比率特征
- `volatility`: 波动率特征
- `volume`: 成交量特征
- `momentum`: 动量特征
- `index`: 索引特征
- `other`: 其他特征

也可以自定义分组：

```python
normalizer = UnifiedNormalizer(
    feature_groups={
        "custom_group1": ["feature1", "feature2"],
        "custom_group2": ["feature3", "feature4"],
    }
)
```

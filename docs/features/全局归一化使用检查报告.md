## ⚠️ 注意事项

### 1. 未来开发建议

在开发新功能时，如果需要使用归一化：

```python
# ✅ 推荐：使用滚动归一化
normalized = normalize_by_group(
    df,
    value_col="feature",
    method="zscore",
    window=252,  # 明确指定窗口大小
)

# ❌ 避免：全局归一化（除非明确需要）
normalized = normalize_by_group(
    df,
    value_col="feature",
    method="zscore",
    window=None,  # 会触发警告
)
```

### 2. 如果确实需要全局归一化

如果确实需要使用全局归一化（如EDA场景），应该：

```python
# 明确关闭警告，表明这是有意的选择
normalized = normalize_by_group(
    df,
    value_col="feature",
    method="zscore",
    window=None,
    warn_global=False,  # 明确关闭警告
)
```

### 3. 推荐配置

- **日频数据**: `window=252`（1年，约252个交易日）
- **周频数据**: `window=52`（1年，52周）
- **小时数据**: `window=168`（1周，7×24小时）
- **分钟数据**: `window=1440`（1天，24×60分钟）

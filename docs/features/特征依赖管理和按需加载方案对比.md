### 方案 1：特征依赖图 + 拓扑排序

#### 优点
- ✅ 依赖关系明确，自动处理
- ✅ 计算顺序自动确定（拓扑排序）
- ✅ 支持按需加载
- ✅ 易于扩展新特征

#### 缺点
- ⚠️ 需要重构现有特征代码（添加装饰器）
- ⚠️ 需要维护依赖关系

#### 适用场景
- 长期项目，需要完整的依赖管理
- 特征数量多，依赖关系复杂
- 需要严格的依赖验证

- ⚠️ 配置和代码可能不同步

#### 适用场景

- 希望配置与代码分离

### 2. 依赖解析算法

```python
def resolve_dependencies(requested_features, registry):
    """
    解析特征依赖关系，返回计算顺序（拓扑排序）
    
    算法：
    1. 收集所有需要的特征（包括依赖）
    2. 构建依赖图
    3. 拓扑排序确定计算顺序
    """
    # 1. 收集所有需要的特征
    all_needed = set(requested_features)
    queue = list(requested_features)
    
    while queue:
        feature = queue.pop(0)
        if feature in registry:
            deps = registry[feature]["dependencies"]
            for dep in deps:
                if dep not in all_needed:
                    all_needed.add(dep)
                    queue.append(dep)
    
    # 2. 构建依赖图
    graph = {f: [] for f in all_needed}
    in_degree = {f: 0 for f in all_needed}
    
    for feature in all_needed:
        if feature in registry:
            deps = registry[feature]["dependencies"]
            for dep in deps:
                if dep in all_needed:
                    graph[dep].append(feature)
                    in_degree[feature] += 1
    
    # 3. 拓扑排序
    queue = [f for f in all_needed if in_degree[f] == 0]
    result = []
    
    while queue:
        feature = queue.pop(0)
        result.append(feature)
        for neighbor in graph[feature]:
            in_degree[neighbor] -= 1
            if in_degree[neighbor] == 0:
                queue.append(neighbor)
    
    # 4. 检查循环依赖
    if len(result) != len(all_needed):
        raise ValueError("Circular dependency detected!")
    
    return result
```

### 3. 特征加载器

```python
class FeatureLoader:
    """
    特征加载器：按需加载特征，自动处理依赖关系
    """
    def __init__(self, registry):
        self.registry = registry
        self.computed_cache = {}  # 缓存已计算的特征
    
    def resolve_dependencies(self, requested_features):
        """解析依赖，返回计算顺序"""
        return resolve_dependencies(requested_features, self.registry)
    
    def load_features(self, df, requested_features, fit=True):
        """
        按需加载特征
        
        Args:
            df: 输入 DataFrame
            requested_features: 请求的特征列表
            fit: 是否拟合（研究阶段=True，实盘阶段=False）
        
        Returns:
            df_with_features: 包含计算特征的 DataFrame
        """
        # 1. 解析依赖
        computation_order = self.resolve_dependencies(requested_features)
        
        # 2. 按顺序计算
        result_df = df.copy()
        for feature_name in computation_order:
            if feature_name in self.computed_cache:
                continue  # 已计算，跳过
            
            feature_info = self.registry[feature_name]
            compute_func = feature_info["compute_func"]
            
            # 检查所需列
            required_cols = feature_info.get("required_columns", [])
            missing_cols = [c for c in required_cols if c not in result_df.columns]
            if missing_cols:
                raise ValueError(
                    f"Missing required columns for {feature_name}: {missing_cols}"
                )
            
            # 计算特征
            result_df = compute_func(result_df, fit=fit)
            
            # 缓存
            self.computed_cache[feature_name] = True
        
        # 3. 只返回请求的特征列
        output_cols = []
        for feature_name in requested_features:
            feature_info = self.registry[feature_name]
            output_cols.extend(feature_info.get("output_columns", [feature_name]))
        
        return result_df[list(df.columns) + output_cols]
```

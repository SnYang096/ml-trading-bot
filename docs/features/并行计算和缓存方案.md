# 并行计算和缓存方案

## 需求

1. **快速并行计算特征**：利用多进程/多线程加速特征计算
2. **利用缓存**：避免重复计算已计算的特征

## 设计思路

### 1. 并行计算策略

#### 按依赖层级并行
- 拓扑排序后，同一层级的特征可以并行计算
- 例如：`atr` 和 `rsi` 无依赖，可以并行计算
- `sqs_hal_high` 依赖 `atr`，需要等 `atr` 计算完成

```
层级 0: [atr, rsi, macd]  → 并行计算
层级 1: [sqs_hal_high, sqs_hal_low]  → 并行计算（依赖层级 0）
层级 2: [sr_strength_max]  → 计算（依赖层级 1）
```

#### 并行实现方式
- **多进程（multiprocessing）**：适合 CPU 密集型计算
- **多线程（threading）**：适合 I/O 密集型计算
- **异步（asyncio）**：适合混合场景

### 2. 缓存策略

#### 内存缓存
- 同一进程内缓存已计算的特征
- 使用 `functools.lru_cache` 或自定义缓存

#### 磁盘缓存
- 持久化缓存，跨进程/跨会话使用
- 使用 `joblib` 或 `pickle` 保存中间结果
- 支持缓存失效和更新

#### 缓存键设计
- 基于特征名 + 数据哈希（前 N 行）
- 或基于特征名 + 参数哈希

## 实现方案

### 1. 依赖层级分析

```python
def analyze_dependency_levels(features: Dict, requested_features: List[str]) -> Dict[int, List[str]]:
    """
    分析特征依赖层级
    
    Returns:
        levels: {level: [feature_names]}
    """
    # 1. 解析依赖
    all_needed = set(requested_features)
    queue = list(requested_features)
    
    while queue:
        feature = queue.pop(0)
        if feature in features:
            deps = features[feature].get("dependencies", [])
            for dep in deps:
                if dep not in all_needed:
                    all_needed.add(dep)
                    queue.append(dep)
    
    # 2. 计算每个特征的层级
    feature_levels = {}
    
    def get_level(feature_name: str) -> int:
        if feature_name in feature_levels:
            return feature_levels[feature_name]
        
        if feature_name not in features:
            return 0
        
        deps = features[feature_name].get("dependencies", [])
        if not deps:
            level = 0
        else:
            level = max([get_level(dep) for dep in deps]) + 1
        
        feature_levels[feature_name] = level
        return level
    
    # 3. 按层级分组
    levels = {}
    for feature in all_needed:
        level = get_level(feature)
        if level not in levels:
            levels[level] = []
        levels[level].append(feature)
    
    return levels
```

### 2. 并行计算器

```python
import multiprocessing as mp
from concurrent.futures import ProcessPoolExecutor, ThreadPoolExecutor, as_completed
from functools import lru_cache
import hashlib
import pickle
import os
from pathlib import Path
from typing import Dict, List, Optional, Callable
import pandas as pd
import numpy as np


class ParallelFeatureComputer:
    """
    并行特征计算器
    
    支持：
    1. 按依赖层级并行计算
    2. 内存缓存
    3. 磁盘缓存
    """
    
    def __init__(
        self,
        cache_dir: Optional[str] = None,
        use_disk_cache: bool = True,
        use_memory_cache: bool = True,
        max_workers: Optional[int] = None,
        parallel_backend: str = "process",  # "process", "thread", "async"
    ):
        """
        Args:
            cache_dir: 磁盘缓存目录
            use_disk_cache: 是否使用磁盘缓存
            use_memory_cache: 是否使用内存缓存
            max_workers: 最大并行数（None 表示使用 CPU 核心数）
            parallel_backend: 并行后端（process/thread/async）
        """
        self.cache_dir = Path(cache_dir) if cache_dir else None
        if self.cache_dir:
            self.cache_dir.mkdir(parents=True, exist_ok=True)
        
        self.use_disk_cache = use_disk_cache
        self.use_memory_cache = use_memory_cache
        self.max_workers = max_workers or mp.cpu_count()
        self.parallel_backend = parallel_backend
        
        # 内存缓存
        self.memory_cache = {}
        
        # 并行执行器
        if parallel_backend == "process":
            self.executor = ProcessPoolExecutor(max_workers=self.max_workers)
        elif parallel_backend == "thread":
            self.executor = ThreadPoolExecutor(max_workers=self.max_workers)
        else:
            self.executor = None  # async 需要特殊处理
    
    def _get_cache_key(self, feature_name: str, df_hash: str, params: Dict) -> str:
        """生成缓存键"""
        params_str = str(sorted(params.items()))
        key_str = f"{feature_name}_{df_hash}_{params_str}"
        return hashlib.md5(key_str.encode()).hexdigest()
    
    def _get_df_hash(self, df: pd.DataFrame, n_rows: int = 100) -> str:
        """生成 DataFrame 哈希（基于前 N 行）"""
        sample = df.head(n_rows)
        # 只使用数值列
        numeric_cols = sample.select_dtypes(include=[np.number]).columns
        sample_data = sample[numeric_cols].values.tobytes()
        return hashlib.md5(sample_data).hexdigest()
    
    def _load_from_disk_cache(self, cache_key: str) -> Optional[pd.DataFrame]:
        """从磁盘加载缓存"""
        if not self.use_disk_cache or not self.cache_dir:
            return None
        
        cache_file = self.cache_dir / f"{cache_key}.pkl"
        if cache_file.exists():
            try:
                with open(cache_file, 'rb') as f:
                    return pickle.load(f)
            except Exception as e:
                print(f"     ⚠️  Error loading cache {cache_key}: {e}")
                return None
        return None
    
    def _save_to_disk_cache(self, cache_key: str, df: pd.DataFrame):
        """保存到磁盘缓存"""
        if not self.use_disk_cache or not self.cache_dir:
            return
        
        cache_file = self.cache_dir / f"{cache_key}.pkl"
        try:
            with open(cache_file, 'wb') as f:
                pickle.dump(df, f)
        except Exception as e:
            print(f"     ⚠️  Error saving cache {cache_key}: {e}")
    
    def _compute_single_feature(
        self,
        feature_name: str,
        feature_info: Dict,
        df: pd.DataFrame,
        fit: bool,
    ) -> tuple[str, pd.DataFrame]:
        """
        计算单个特征
        
        Returns:
            (feature_name, df_with_feature)
        """
        # 1. 检查内存缓存
        if self.use_memory_cache and feature_name in self.memory_cache:
            print(f"     💾 Using memory cache for {feature_name}")
            return (feature_name, self.memory_cache[feature_name])
        
        # 2. 检查磁盘缓存
        df_hash = self._get_df_hash(df)
        compute_params = feature_info.get("compute_params", {})
        cache_key = self._get_cache_key(feature_name, df_hash, compute_params)
        
        cached_df = self._load_from_disk_cache(cache_key)
        if cached_df is not None:
            print(f"     💾 Using disk cache for {feature_name}")
            if self.use_memory_cache:
                self.memory_cache[feature_name] = cached_df
            return (feature_name, cached_df)
        
        # 3. 计算特征
        compute_func_name = feature_info["compute_func"]
        compute_func = get_compute_func(compute_func_name)
        
        try:
            if compute_params:
                result_df = compute_func(df, **compute_params)
            else:
                result_df = compute_func(df)
            
            # 4. 保存缓存
            if self.use_memory_cache:
                self.memory_cache[feature_name] = result_df
            
            if self.use_disk_cache:
                self._save_to_disk_cache(cache_key, result_df)
            
            print(f"     ✅ Computed {feature_name}")
            return (feature_name, result_df)
        except Exception as e:
            print(f"     ❌ Error computing {feature_name}: {e}")
            return (feature_name, df)  # 返回原始 DataFrame
    
    def compute_features_parallel(
        self,
        df: pd.DataFrame,
        features: Dict,
        requested_features: List[str],
        fit: bool = True,
    ) -> pd.DataFrame:
        """
        并行计算特征
        
        Args:
            df: 输入 DataFrame
            features: 特征配置字典
            requested_features: 请求的特征列表
            fit: 是否拟合
        
        Returns:
            df_with_features: 包含计算特征的 DataFrame
        """
        # 1. 分析依赖层级
        levels = analyze_dependency_levels(features, requested_features)
        
        print(f"   📊 Computing {len(requested_features)} features in {len(levels)} levels...")
        
        result_df = df.copy()
        
        # 2. 按层级顺序计算（每层内并行）
        for level in sorted(levels.keys()):
            level_features = levels[level]
            print(f"   🔄 Level {level}: Computing {len(level_features)} features in parallel...")
            
            # 提交并行任务
            futures = []
            for feature_name in level_features:
                if feature_name not in features:
                    continue
                
                feature_info = features[feature_name]
                future = self.executor.submit(
                    self._compute_single_feature,
                    feature_name,
                    feature_info,
                    result_df,
                    fit,
                )
                futures.append(future)
            
            # 等待所有任务完成
            for future in as_completed(futures):
                feature_name, feature_df = future.result()
                
                # 合并结果（只合并新增的列）
                new_cols = [c for c in feature_df.columns if c not in result_df.columns]
                if new_cols:
                    result_df = pd.concat([result_df, feature_df[new_cols]], axis=1)
        
        return result_df
    
    def clear_cache(self, memory: bool = True, disk: bool = False):
        """清除缓存"""
        if memory:
            self.memory_cache.clear()
            print("   🗑️  Memory cache cleared")
        
        if disk and self.cache_dir:
            for cache_file in self.cache_dir.glob("*.pkl"):
                cache_file.unlink()
            print("   🗑️  Disk cache cleared")
    
    def __del__(self):
        """清理资源"""
        if self.executor:
            self.executor.shutdown(wait=True)
```

### 3. 集成到 StrategyFeatureLoader

```python
class StrategyFeatureLoader:
    """
    基于纯配置文件的特征加载器（支持并行计算和缓存）
    """
    
    def __init__(
        self,
        feature_deps_path: str = "config/feature_dependencies.yaml",
        strategy_config_path: str = "config/strategy_features.yaml",
        cache_dir: Optional[str] = "cache/features",
        use_disk_cache: bool = True,
        use_memory_cache: bool = True,
        max_workers: Optional[int] = None,
        parallel_backend: str = "process",
    ):
        """
        初始化特征加载器
        
        Args:
            feature_deps_path: 特征依赖配置文件路径
            strategy_config_path: 策略配置文件路径
            cache_dir: 磁盘缓存目录
            use_disk_cache: 是否使用磁盘缓存
            use_memory_cache: 是否使用内存缓存
            max_workers: 最大并行数
            parallel_backend: 并行后端（process/thread）
        """
        self.feature_deps = self._load_yaml(feature_deps_path)
        self.strategy_config = self._load_yaml(strategy_config_path)
        
        # 创建并行计算器
        self.computer = ParallelFeatureComputer(
            cache_dir=cache_dir,
            use_disk_cache=use_disk_cache,
            use_memory_cache=use_memory_cache,
            max_workers=max_workers,
            parallel_backend=parallel_backend,
        )
    
    def load_strategy_features(
        self,
        df: pd.DataFrame,
        strategy_name: str,
        fit: bool = True,
    ) -> pd.DataFrame:
        """
        为指定策略加载特征（支持并行计算和缓存）
        
        Args:
            df: 输入 DataFrame
            strategy_name: 策略名称
            fit: 是否拟合
        
        Returns:
            df_with_features: 包含计算特征的 DataFrame
        """
        if strategy_name not in self.strategy_config["strategies"]:
            raise ValueError(f"Unknown strategy: {strategy_name}")
        
        strategy_config = self.strategy_config["strategies"][strategy_name]
        result_df = df.copy()
        
        # 1. 生成基础特征（如果需要）
        if strategy_config.get("base_feature_types"):
            print(f"   🔧 Generating base features ({strategy_config['base_feature_types']})...")
            engineer = ComprehensiveFeatureEngineer(
                feature_types=",".join(strategy_config["base_feature_types"])
            )
            result_df = engineer.engineer_all_features(result_df, fit=fit)
        
        # 2. 并行计算策略专属特征
        requested_features = strategy_config["requested_features"]
        features = self.feature_deps["features"]
        
        result_df = self.computer.compute_features_parallel(
            result_df,
            features,
            requested_features,
            fit=fit,
        )
        
        # 3. 只返回请求的特征列
        output_cols = []
        for feature_name in requested_features:
            if feature_name in features:
                feature_info = features[feature_name]
                output_cols.extend(feature_info.get("output_columns", [feature_name]))
        
        all_cols = list(df.columns) + [c for c in output_cols if c in result_df.columns]
        return result_df[all_cols]
    
    def clear_cache(self, memory: bool = True, disk: bool = False):
        """清除缓存"""
        self.computer.clear_cache(memory=memory, disk=disk)
```

## 性能优化建议

### 1. 并行后端选择

- **CPU 密集型**（如 WPT、Hurst）：使用 `process`（多进程）
- **I/O 密集型**（如读取文件）：使用 `thread`（多线程）
- **混合场景**：使用 `process`（默认）

### 2. 缓存策略

- **内存缓存**：适合同一会话内重复计算
- **磁盘缓存**：适合跨会话、跨进程使用
- **缓存键优化**：基于数据哈希，避免数据变化时使用旧缓存

### 3. 批量计算优化

- **按层级批量提交**：减少任务提交开销
- **结果合并优化**：只合并新增列，避免重复复制

## 注意事项

1. **线程安全**：确保计算函数是线程安全的
2. **内存管理**：大量并行计算可能占用大量内存
3. **缓存失效**：数据变化时需要清除缓存
4. **错误处理**：单个特征计算失败不应影响其他特征

## 总结

通过并行计算和缓存机制，可以显著提升特征计算速度：

1. **并行计算**：按依赖层级并行，充分利用多核 CPU
2. **内存缓存**：同一会话内快速复用
3. **磁盘缓存**：跨会话持久化，避免重复计算
4. **灵活配置**：支持不同的并行后端和缓存策略

预计性能提升：**2-10倍**（取决于特征数量和 CPU 核心数）

## 🔄 迁移指南

### 如果您的代码使用旧模块

#### 情况1: 直接导入 UnifiedNormalizer

**旧代码**:
```python
from src.data_tools.unified_normalization import UnifiedNormalizer
```

**新代码**:
```python
from src.features.time_series.utils_normalization import UnifiedNormalizer
```

**注意**: ⚠️ 旧导入路径已不可用，必须使用新路径。

#### 情况2: 使用旧参数

**旧代码**:
```python
normalizer = UnifiedNormalizer(
    strategy="rolling",
    scaler_type="standard",
    window_size=100,
)
```

**新代码**:
```python
normalizer = UnifiedNormalizer(
    method="zscore",  # scaler_type="standard" → method="zscore"
    window=252,  # window_size → window，推荐值改为252
    group_col="_symbol",  # 新增：多资产支持
)
```

#### 参数映射表

| 旧参数 | 新参数 | 说明 |
|--------|--------|------|
| `strategy` | 已移除 | 统一使用滚动归一化 |
| `scaler_type="standard"` | `method="zscore"` | |
| `scaler_type="minmax"` | `method="minmax"` | |
| `scaler_type="robust"` | `method="robust_minmax"` | |
| `window_size` | `window` | 默认值改为252 |

## ⚠️ 重要变化

### 1. 默认行为变化

**旧模块**:
- 默认 `window_size=100`
- 需要手动指定滚动归一化

**新模块**:
- 默认 `window=252`（推荐值，防止未来信息泄露）
- 自动使用滚动归一化

### 2. 多资产支持

**新模块**新增 `group_col` 参数，支持多资产归一化：

```python
normalizer = UnifiedNormalizer(
    method="zscore",
    window=252,
    group_col="_symbol",  # 按资产分组归一化
)
```

### 3. 未来信息泄露警告

**新模块**在使用全局归一化（`window=None`）时会自动警告：

```python
# ⚠️ 会收到警告
normalizer = UnifiedNormalizer(window=None)
```

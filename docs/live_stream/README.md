# 实时流计算（权威入口）

**状态**: ✅ 当前版本  
**最后更新**: 2026-01  
**相关文档**: [主文档索引](../README.md)

本目录用于**统一说明**本项目在**事件回测**与**实盘**下的实时数据流：数据接入 → 事件流 → 特征计算 → 存储 → 补全/对账 → 异常发现与自愈，目标是：

- **回测与实盘一致**：同一套事件模型、同一套特征契约、同一套异常处理策略。
- **实盘稳定**：可恢复（重连/补全/重放）、可观测（质量监控/告警）、可降级（熔断/跳过交易/跳过部分特征）。

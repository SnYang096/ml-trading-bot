## 🔍 故障排除

如果测试失败：

1. **检查 Docker**:
   ```bash
   docker ps
   ```

2. **查看详细日志**:
   ```bash
   make test-complex-features-comprehensive 2>&1 | tee test.log
   ```

3. **检查依赖**:
   - 确保 Docker 镜像包含所有必需的包
   - 如果需要，重新构建镜像: `make docker-build`

4. **单独运行测试**:
   ```bash
   # 只运行一个测试类
   docker run --rm -v $(pwd):/workspace -w /workspace \
     hansenlovefiona017/lightgbm-runtime:v0.0.5 \
     python3 -m pytest tests/test_complex_features_comprehensive.py::TestGARCHMultiAsset -v
   ```

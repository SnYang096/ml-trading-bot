🛠️ 你提到的两个例子，恰恰是顶级系统的做法：
✅ 例1：range_bound → 波动率绝对收益
在震荡市中，方向性 alpha 枯竭，但波动率套利机会增多（如期权 skew、gamma squeeze）
可切换至：
做多 realized vol / 做空 implied vol（如果有衍生品）
或用 mean-reversion 信号 + 波动率缩放仓位
本质仍是“绝对收益”，但信号逻辑从 trend → reversion
📌 这不是“换策略”，而是同一目标（绝对收益）下的信号源切换。

✅ 例2：strong_trend → 多空不平衡横截面
在强趋势中，若仍强制 50/50 long-short，会浪费 beta 机会
可动态调整：
net exposure = f(trend_strength)
例如：BTC 趋势得分 > 0.8 → 允许 net long 70%
此时横截面模型仍在工作（选最强/最弱），但组合不再中性
📌 这不是“放弃超额收益”，而是在保留 alpha 的同时捕获部分 beta。

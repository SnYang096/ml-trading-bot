## 🔧 具体修改内容

### 加权投票逻辑
```python
# 各层权重
strategic_weight = 0.5  # 战略层权重最高
tactical_weight = 0.3   # 战术层权重中等
execution_weight = 0.2  # 执行层权重较低

# 投票阈值
vote_threshold = 0.4    # 至少40%权重支持才能交易

# 各层最小置信度（降低门槛）
strategic_min_confidence = 0.2   # 战略层最低20%
tactical_min_confidence = 0.1    # 战术层最低10%
execution_min_confidence = 0.05  # 执行层最低5%
```

### 融合决策流程
1. **计算各层得分**: `confidence × weight`
2. **计算总得分**: `strategic_score + tactical_score + execution_score`
3. **投票决策**: `total_score / max_possible_score >= vote_threshold`
4. **最终置信度**: 加权平均

  def test_feature_correlation_reasonableness():

#### 3. **lag 特征有效性衰减测试** ❌
- **文档要求**: lag 特征有效性应缓慢衰减，而非断崖下降
- **建议实现**:
  ```python
  def test_lag_effect_decay_smoothly():
      """测试 lag 特征有效性应缓慢衰减"""
      features = extract_features(df)
      
      # 构造简单目标（仅用于测试）
      log_ret = np.log(df["close"] / df["close"].shift(1))
      target = log_ret.rolling(5).std().shift(-5)
      
      lag_cols = ["vol_lag_1", "vol_lag_2", "vol_lag_3"]
      correlations = []
      for col in lag_cols:
          if col in features.columns:
              corr = features[col].corr(target, min_periods=50)
              correlations.append(abs(corr))
      
      # 应大致递减，且无断崖
      if len(correlations) >= 2:
          assert correlations[1] > 0.1 * correlations[0], "lag_2 相关性断崖式下降"
  ```

## 二、现有测试的优秀实践

### 1. **测试数据生成模式**
```python
def create_mock_data(n_samples: int = 500, seed: int = 42) -> pd.DataFrame:
    """创建模拟数据用于测试"""
    np.random.seed(seed)
    dates = pd.date_range("2024-01-01", periods=n_samples, freq="5min")
    
    # 生成价格数据（随机游走 + 趋势）
    returns = np.random.randn(n_samples) * 0.01
    trend = np.sin(np.arange(n_samples) / 50) * 0.001
    volatility_cluster = np.abs(np.random.randn(n_samples)) * 0.005
    returns = returns + trend + volatility_cluster
    prices = 100 * np.exp(np.cumsum(returns))
    
    return pd.DataFrame({"close": prices}, index=dates)
```

def create_multi_asset_data():

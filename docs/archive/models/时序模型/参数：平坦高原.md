def local_flatness(performance, window=3):
"""计算滑动窗口内的标准差，越小越平坦"""

python
def plateau_ratio(performance, tolerance=0.1):
"""性能在 top (1-tol) 区间的参数占比"""
max_perf = max(performance)
threshold = max_perf (1 - tolerance) # 如 tol=0.1 → 90% 以上算高原
plateau_count = sum(p >= threshold for p in performance)
return plateau_count / len(performance)

ratio = plateau_ratio(calmar, tolerance=0.1) # >0.6 可认为是高原

@ 平坦高原原理

“平坦高原”（Flat Plateau）是量化策略开发中一个核心稳健性判据，它背后融合了统计学习理论、优化稳定性、以及实盘泛化能力的思想。下面我从原理、数学本质、实战意义三个层面为你系统解析。

🧠 一、直观理解：什么是“平坦高原”？

想象你在一座山里找最高点：
尖锐峰值（Sharp Peak）：
只有一个精确坐标（如参数=24.0）能登顶
脚稍微偏一点（23.9 或 24.1），高度骤降
→ 对噪声极度敏感，实盘极易失效
平坦高原（Flat Plateau）：
一片广阔区域（如参数∈[22, 28]）都接近最高点
无论你站在哪，视野（策略表现）几乎一样好
→ 对参数扰动鲁棒，实盘稳定
✅ 平坦高原 = 策略的“容错区”
它意味着：你的策略不是靠“猜中某个神奇数字”赚钱，而是捕捉到了真实的市场规律。

📐 二、数学原理：为什么平坦高原代表泛化能力？
1. 来自统计学习理论：平坦极小值 vs 尖锐极小值

在机器学习中，Hochreiter & Schmidhuber (1997) 提出：
“平坦的损失极小值（flat minima）比尖锐的极小值（sharp minima）具有更好的泛化能力。”

虽然你做的是规则策略而非神经网络，但思想完全相通：

概念映射 机器学习 量化策略
-------- -------- --------
损失函数 Training Loss 回测负收益（如 -Calmar）
参数 网络权重 MA窗口、仓位系数等
泛化误差 Test Loss 实盘 vs 回测差距
平坦极小值：在参数空间中，损失函数在一个大区域内变化平缓
→ 对参数扰动不敏感 → 低泛化误差
尖锐极小值：仅在一个极小邻域内最优
→ 微小扰动导致性能崩溃 → 高过拟合风险
2. 信息论视角：平坦高原 = 低复杂度模型

根据 Minimum Description Length (MDL) 原则：
能用更少信息描述的模型（即对参数不敏感），更可能抓住本质规律。
如果你必须精确指定 “MA=24.0037” 才能赚钱 → 模型复杂度过高
如果 “MA∈[20,30] 都行” → 模型简洁，大概率学到真信号
3. 贝叶斯解释：平坦区域对应高后验概率

在贝叶斯优化中，目标函数的后验分布在平坦区域有更大的概率质量。
这意味着：即使未来市场略有漂移，策略仍大概率有效。

⚙️ 三、为什么尖锐峰值通常是“过拟合噪声”？
根本原因：历史数据有限 + 市场非平稳
你回测的数据只是真实市场的一个样本路径
噪声（如某次插针、交易所故障）会被某些参数“巧合拟合”
这些参数在其他样本路径（未来）中失效
🌰 举例：
某策略在 window=24 时完美避开 2022 年 LUNA 崩盘，
但在 window=25 时没躲开 → 性能断崖。
这不是策略强，而是恰好拟合了一次黑天鹅的时间点。

而平坦高原策略：
在 window∈[20,30] 都能部分规避风险 → 说明它真正识别了“高波动预警”信号，而非记住日期。

📊 四、如何从数学上刻画“平坦度”？
1. 一阶条件：梯度接近零
在最优参数 \( \theta^ \) 附近，若：
\[
\nabla_\theta \mathcal{L}(\theta) \approx 0 \quad \text{for all } \theta \in [\theta^ - \delta, \theta^ + \delta]
\]
→ 说明是平坦区域
2. 二阶条件：Hessian 特征值小
Hessian 矩阵 \( H = \nabla^2 \mathcal{L}(\theta^) \) 的特征值 \( \lambda_i \) 反映曲率：
若所有 \( \lambda_i \ll 1 \) → 平坦
若存在 \( \lambda_i \gg 1 \) → 尖锐
💡 在单参数情况下，就是看 二阶导数（曲率）是否接近 0
3. 实用代理指标（无需求导）
由于策略回测不可导，我们用：
局部方差：\( \text{Var}(\text{perf} \mid \theta \in \text{neighborhood}) \)
高原占比：\( \frac{ \{\theta : \text{perf}(\theta) \geq 0.95 \cdot \max \text{perf}\} }{ \Theta } \)

🛡️ 五、平坦高原的实战价值

维度 尖锐峰值策略 平坦高原策略
------ ------------ ------------
实盘稳定性 极差（参数漂移即失效） 强（容忍市场变化）
参数调优难度 需暴力搜索高精度 粗粒度即可
可解释性 像“玄学”（为何是24？） 可解释（20~30都有效）
维护成本 高频重调参 低频微调
监管/合规 难以证明稳健性 易通过压力测试
🎯 机构级策略的底线要求：关键参数必须落在平坦高原。

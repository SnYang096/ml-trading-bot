均值回归（Mean Reversion）策略的核心假设是：价格偏离“合理水平”后，会向均值靠拢。因此，特征工程的关键是 识别“过度偏离”和“回归动能”。

以下是经过实战验证的 均值回归特征选择方法，按逻辑模块分类，并附代码示例（Python + TA-Lib / Pandas）：

🧠 一、核心思想：均值回归特征 = 偏离度 + 回归势能 + 环境稳定性

模块 目标 典型特征
------ ------ --------
1. 偏离度（Overextension） 衡量当前价格是否“太远” 布林带宽度、Z-Score、RSI、偏离均线百分比
2. 回归势能（Reversion Momentum） 判断回调是否启动 反转K线形态、动量衰减、成交量萎缩
3. 环境稳定性（Regime Filter） 确保处于震荡市 波动率低位、ADX低、Hurst指数<0.5
4. 时间尺度（Lookback Window） 匹配回归周期 多周期组合（短/中/长）

🔢 二、具体特征清单（附代码）
✅ 1. 标准化偏离度（最核心）
这些特征直接量化“价格离均值有多远”。

python
import numpy as np
import pandas as pd
import talib
(1) Z-Score (滚动标准化)
def zscore(series, window=20):
return (series - series.rolling(window).mean()) / series.rolling(window).std()

X['zscore_20'] = zscore(close, 20)
X['zscore_50'] = zscore(close, 50)
(2) 布林带位置（Bollinger Band Position）
upper, mid, lower = talib.BBANDS(close, timeperiod=20, nbdevup=2, nbdevdn=2)
X['bb_position'] = (close - lower) / (upper - lower) # 0~1，>0.8 超买，<0.2 超卖
(3) 偏离均线百分比
X['pct_above_ma20'] = close / talib.SMA(close, 20) - 1
X['pct_above_ma50'] = close / talib.SMA(close, 50) - 1
(4) RSI（超买超卖）
X['rsi_14'] = talib.RSI(close, 14)
X['rsi_21'] = talib.RSI(close, 21)
💡 关键技巧：不要只用单一窗口！组合多个周期（如 10/20/50）捕捉不同尺度的偏离。

✅ 2. 回归动能信号（确认反转开始）
避免“接飞刀”——价格可能继续远离均值。这些特征判断 回归是否启动。

python
(5) 动量衰减（Momentum Divergence）
mom_10 = talib.MOM(close, 10)
X['mom_decay'] = mom_10.diff() < 0 # 动量开始下降
(6) 反转K线形态（TA-Lib）
X['hammer'] = talib.CDLHAMMER(open_, high, low, close) != 0
X['engulfing'] = talib.CDLENGULFING(open_, high, low, close) != 0
(7) 成交量萎缩（Volatility Contraction）
vol_ma = volume.rolling(20).mean()
X['low_volume'] = volume < vol_ma 0.8 # 低成交量配合价格极端 → 易反转
(8) 价格加速度（二阶导）
X['price_accel'] = close.diff().diff() # 加速度为负 → 上涨减速

✅ 3. 市场状态过滤器（只在震荡市交易）
这是均值回归成败的关键！ 在趋势市中做均值回归 = 逆势扛单。

python
(9) ADX（趋势强度）
adx = talib.ADX(high, low, close, 14)
X['adx_low'] = adx < 20 # ADX<20 表示无趋势
(10) 波动率分位数（低波动环境）
volatility = close.pct_change().rolling(20).std()
X['vol_quintile'] = pd.qcut(volatility, 5, labels=False, duplicates='drop')
(11) Hurst 指数（长期记忆性）
def hurst(ts, max_lag=20):
lags = range(2, min(max_lag, len(ts)//2))
if len(lags) < 2: return 0.5
tau = [np.sqrt(np.std(np.subtract(ts[lag:], ts[:-lag]))) for lag in lags]
poly = np.polyfit(np.log(lags), np.log(tau), 1)
return poly[0]2.0
滚动计算（较慢，可每N根K线算一次）
X['hurst_100'] = close.rolling(100).apply(lambda x: hurst(x), raw=False)
H < 0.5 → 均值回归；H > 0.5 → 趋势
(12) 布林带宽度（Band Width）
band_width = (upper - lower) / mid
X['bb_narrow'] = band_width < band_width.rolling(50).quantile(0.3)
⚠️ 强烈建议：将 adx_low 或 hurst < 0.5 作为 样本过滤条件，而非普通特征。

✅ 4. 多时间框架融合
单一周期容易假信号。组合多周期提升鲁棒性。

python
(13) 多周期Z-Score一致性
X['zscore_consensus'] = (
(zscore(close, 10) > 1.5).astype(int) +
(zscore(close, 20) > 1.5).astype(int) +
(zscore(close, 50) > 1.0).astype(int)
) # 值越大，多周期一致超买
(14) 短期 vs 长期偏离方向
X['short_over_long'] = zscore(close, 10) - zscore(close, 50)
0：短期更极端，可能先回调

📊 四、典型均值回归特征组合（推荐起点）

类别 推荐特征
------ --------
核心偏离 zscore_20, bb_position, rsi_14
动能确认 mom_decay, price_accel
状态过滤 adx_low（用于样本过滤）, vol_quintile
多周期 zscore_consensus
✅ 初始特征集控制在 8~12 个，避免维度灾难。

⚠️ 五、常见陷阱

1. 在趋势市强行做均值回归
→ 必须用 ADX/Hurst 过滤样本！

2. 只用单一指标（如只看 RSI）
→ RSI 在趋势市中可长期超买/超卖，需结合价格结构。

3. 忽略交易成本
→ 均值回归盈利空间小，高频交易需考虑滑点。

4. 回测过拟合
→ 使用 滚动窗口 OOS 测试，而非单次划分。

🛠 六、完整示例：构建均值回归特征集

python
def build_mean_reversion_features(df):
close = df['close']
high = df['high']
low = df['low']
volume = df['volume']

X = pd.DataFrame(index=df.index)

# 偏离度
X['zscore_20'] = zscore(close, 20)
upper, _, lower = talib.BBANDS(close, 20, 2, 2)
X['bb_position'] = (close - lower) / (upper - lower)
X['rsi_14'] = talib.RSI(close, 14)

# 动能
mom = talib.MOM(close, 10)
X['mom_decay'] = (mom.diff() < 0).astype(int)
X['price_accel'] = close.diff().diff()

# 状态（用于后续过滤）
X['adx'] = talib.ADX(high, low, close, 14)

# 多周期
X['zscore_10'] = zscore(close, 10)
X['zscore_consensus'] = ((X['zscore_10'] > 1.5) & (X['zscore_20'] > 1.2)).astype(int)

return X

✅ 总结

步骤 关键动作
------ --------
1. 构造特征 覆盖 偏离度 + 动能 + 状态
2. 过滤样本 只在震荡市训练（ADX<20 或 Hurst<0.5）
3. 筛选特征 用 IC/SHAP 保留强信号
4. 验证逻辑 检查 SHAP：超买时 RSI/zscore 应为负贡献
🎯 记住：均值回归不是“猜顶摸底”，而是 在高概率回归环境中，捕捉已启动的回调。

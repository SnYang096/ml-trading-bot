# 特征选择关键词机制说明

## 概述

`reversal_keywords`、`breakout_keywords`、`compression_keywords` 是用于**自动筛选特征**的关键词列表。

## 核心作用

### 问题背景

在特征工程完成后，我们可能有数百甚至数千个特征，但并不是所有特征都适合每个模型：

- **反转型模型**：需要 SR 强度、反转信号等特征
- **突破型模型**：需要突破动量、趋势延续等特征
- **压缩突破模型**：需要压缩指标、突破动量等特征

如果让每个模型使用所有特征，会导致：
1. **特征噪声**：无关特征引入噪声
2. **模型混淆**：模型难以学习到正确的模式
3. **计算浪费**：训练时间和内存浪费

### 解决方案：关键词匹配筛选

通过定义关键词列表，自动从所有特征中筛选出相关的特征。

## 工作原理

### 1. `reversal_keywords` - 反转型模型特征筛选

```python
reversal_keywords = [
    # SR 区域相关
    "sqs", "dist_to_nearest_sr", "sr_density", "sr_confluence",
    "nearest_sr", "direction_to_nearest_sr",
    # 反转信号相关
    "reversal", "reversed", "fake_breakout", "breakout_status",
    "price_reversed_before_sr",
    # SR 边界强度
    "sr_strength", "boundary_strength", "support_sqs", "resistance_sqs",
    # 成交量确认
    "volume_ratio", "vol_ratio", "volume_confirmation",
    # 压缩和结构
    "compression", "compression_score", "compression_confidence",
    # 多时间框架
    "trend_context", "trend_4h", "volatility_regime",
    # 边界质量
    "breakout_quality", "breakout_confirmation", "role_flip",
    # 价格行为
    "price_action", "candlestick",
    # Default 特征（技术指标，可能对反转有用）
    "rsi", "macd", "stochastic", "williams",
]

# 筛选逻辑
selected = []
for feat in all_features:
    feat_lower = feat.lower()  # 转小写，便于匹配
    # 检查特征名称是否包含任何关键词
    if any(keyword in feat_lower for keyword in reversal_keywords):
        selected.append(feat)  # 包含关键词 → 选中
```

### 2. 匹配示例

假设我们有这些特征：
- `sqs_hal_high` → 包含 "sqs" → ✅ 选中
- `dist_to_nearest_sr` → 包含 "dist_to_nearest_sr" → ✅ 选中
- `volume_ratio_20` → 包含 "volume_ratio" → ✅ 选中
- `breakout_speed` → 不包含任何关键词 → ❌ 不选中（排除）
- `rsi_14` → 包含 "rsi" → ✅ 选中
- `macd_signal` → 包含 "macd" → ✅ 选中
- `momentum_decay` → 不包含任何关键词 → ❌ 不选中（排除）

### 3. 完整筛选逻辑

```python
def select_reversal_features(df: pd.DataFrame, all_features: List[str]) -> List[str]:
    reversal_keywords = [...]
    
    selected = []
    for feat in all_features:
        feat_lower = feat.lower()
        
        # 1. 检查是否包含关键词 → 选中
        if any(keyword in feat_lower for keyword in reversal_keywords):
            selected.append(feat)
        
        # 2. 排除突破相关特征
        elif "breakout_speed" in feat_lower or "momentum_decay" in feat_lower:
            continue  # 这些更适合突破型
        
        # 3. 保留通用特征
        else:
            if any(keyword in feat_lower for keyword in ["atr", "volatility", "vol", "trend_strength"]):
                selected.append(feat)
    
    return selected
```

## 为什么使用关键词匹配？

### 优点

1. **自动化**：不需要手动列出每个特征名称
2. **灵活性**：新特征只要名称包含关键词就会被自动选中
3. **可维护性**：只需维护关键词列表，不需要维护完整的特征列表
4. **可扩展性**：可以轻松添加新的关键词

### 缺点

1. **可能误选**：如果特征名称恰好包含关键词但不相关
2. **可能漏选**：如果特征名称不包含关键词但实际相关
3. **依赖命名规范**：需要特征命名有一定的规范性

### 改进建议

1. **精确匹配**：对于关键特征，可以使用精确匹配
2. **特征重要性分析**：训练后分析特征重要性，动态调整
3. **特征白名单**：对于关键特征，维护一个白名单

## 关键词分类说明

### 1. SR 区域相关关键词

```python
"sqs", "dist_to_nearest_sr", "sr_density", "sr_confluence",
"nearest_sr", "direction_to_nearest_sr"
```

**匹配的特征示例：**
- `sqs_hal_high` ✅
- `sqs_zigzag_low` ✅
- `dist_to_nearest_sr` ✅
- `sr_density_20` ✅
- `sr_confluence_score` ✅

**为什么重要：**
- 反转型模型需要知道 SR 区域的强度和质量
- 这些特征直接反映 SR 区域的有效性

### 2. 反转信号相关关键词

```python
"reversal", "reversed", "fake_breakout", "breakout_status",
"price_reversed_before_sr"
```

**匹配的特征示例：**
- `price_reversed_before_sr` ✅
- `fake_breakout_count` ✅
- `breakout_status` ✅
- `reversal_signal` ✅

**为什么重要：**
- 这些特征直接标识反转行为
- 是反转型模型的核心特征

### 3. 成交量确认关键词

```python
"volume_ratio", "vol_ratio", "volume_confirmation"
```

**匹配的特征示例：**
- `volume_ratio_20` ✅
- `vol_ratio_5` ✅
- `volume_confirmation_score` ✅

**为什么重要：**
- 反转时成交量应该萎缩或异常放大
- 用于确认反转的有效性

### 4. 技术指标关键词（Default 特征）

```python
"rsi", "macd", "stochastic", "williams"
```

**匹配的特征示例：**
- `rsi_14` ✅
- `macd_signal` ✅
- `stochastic_k` ✅
- `williams_r` ✅

**为什么重要：**
- RSI、Stochastic、Williams 都是超买/超卖指标
- 对反转信号有很好的指示作用
- MACD 背离也是反转信号

## 总结

`reversal_keywords` 的作用是：

1. **自动筛选特征**：从所有特征中筛选出与反转型相关的特征
2. **提高模型质量**：只使用相关特征，减少噪声
3. **提高训练效率**：减少特征数量，加快训练速度
4. **保持灵活性**：新特征只要名称包含关键词就会被自动选中

这是一个**基于特征名称的自动筛选机制**，通过关键词匹配来决定哪些特征适合哪个模型。

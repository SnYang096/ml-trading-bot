## 🧩 二、为什么说 BOS/CHoCH “计算并不复杂”？

很多人觉得“结构分析”是主观的，其实完全可量化。

### ✅ BOS（Break of Structure）量化定义：
```python
# 1H 图
if current_high > previous_high and previous_high > previous_low:
    bullish_bos = True  # 上升趋势延续
```

### ✅ CHoCH（Change of Character）量化定义：
```python
# 原下跌趋势：不断创新低
# 当前K线：不再创新低，且收盘 > 前K线高点
if not new_low and close > previous_high:
    potential_choch = True  # 可能反转
```

👉 **这些只是简单的 if 判断，计算成本极低**，远低于 EMA 的滚动计算。

## 🎯 四、为什么均线“看似简单”，实则不适合 `intraday_sniper`？

### ❌ 均线的三大硬伤：
1. **滞后性**：EMA 是平均值，天然滞后于价格。
2. **震荡市陷阱**：在区间行情中，均线反复交叉，导致过度交易。
3. **无法识别“结构反转”**：它只告诉你“在上面还是下面”，但不告诉你“趋势是否已变”。

> 而 `intraday_sniper` 的目标是：**在趋势启动的瞬间狙击，避免在震荡中损耗**。  
> 均线做不到前者，却极易导致后者。

## ✅ 五、那均线就没用了吗？

**不是**。均线仍有价值，但角色不同：

### 推荐用法：
- ✅ **作为辅助确认**：BOS 发生时，若价格也在 50EMA 之上，趋势更健康。
- ✅ **作为动态支撑**：上升趋势中，50EMA 可作为回踩加仓参考。
- ✅ **多时间框架共振**：1H BOS + 15m 50EMA 向上 → 信号更强。

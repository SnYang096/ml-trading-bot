## 🧩 二、可增强的数学工具总览（按功能分类）

### 1️⃣ **时频域分析类**（改进小波）

| 工具                                           | 作用                | 增强效果                |
| -------------------------------------------- | ----------------- | ------------------- |
| **Hilbert–Huang Transform (HHT)**            | 适合非线性非平稳信号，提取瞬时频率 | 比小波更敏感于波动结构变化       |
| **Empirical Mode Decomposition (EMD)**       | 自适应分解信号成IMF分量     | 取代固定小波尺度，更符合市场的多节奏性 |
| **短时傅里叶变换 (STFT)**                           | 固定窗口时频分析          | 可用于小波能量验证           |
| **Synchrosqueezed Wavelet Transform (SSWT)** | 提高时频聚焦度的小波增强      | 高频扩张识别精准提升30%       |

👉 小波是基础，但 **EMD / HHT / SSWT** 是它的进化版，可构建“**自适应能量谱系统**”。

---

### 2️⃣ **几何与结构分析类**

| 工具                                 | 功能         | 增强方向               |
| ---------------------------------- | ---------- | ------------------ |
| **分形维度分析 (Fractal Dimension)**     | 测量价格路径的粗糙度 | 提取趋势持续性、结构复杂度      |
| **拓扑持久性 (Persistent Homology)**    | 识别价格流形形状变化 | 自动发现结构破坏（结构失败止损信号） |
| **形态滤波 (Morphological Filtering)** | 图像学思想在K线应用 | 去除虚假顶底，保留关键转折结构    |

这些方法可让系统从“几何趋势”升级为“拓扑趋势”，识别趋势连续性和结构完整性。

---

### 3️⃣ **动态系统与状态估计类**

| 工具                                  | 功能          | 应用方向                       |
| ----------------------------------- | ----------- | -------------------------- |
| **Kalman Filter / Particle Filter** | 动态信号平滑与状态估计 | 自适应趋势、动能平滑与噪声隔离            |
| **Hidden Markov Model (HMM)**       | 市场状态转移建模    | 可替代手工的压缩/扩展判定，形成概率版“市场状态机” |
| **Dynamic Bayesian Network (DBN)**  | 时间因果链建模     | 对订单流 → 价格反应延迟的学习           |

这些工具让系统具备**状态感知与预测性**，不是“看见结构”，而是“预测结构变化”。

---

### 4️⃣ **信息论与统计物理类**

| 工具                           | 功能           | 应用方向                     |
| ---------------------------- | ------------ | ------------------------ |
| **熵 (Entropy)**              | 度量市场信息量与混沌程度 | 构建“压缩熵指标”：低熵=压缩，高熵=扩张    |
| **互信息 (Mutual Information)** | 度量指标间非线性相关性  | 判断订单流与价格的真实关联            |
| **KL散度 / JS散度**              | 分布偏离检测       | 用于检测价格结构是否发生“分布突变”       |
| **熵流 (Entropy Flow)**        | 市场信息流方向      | 可识别多周期能量传导方向（短周期向长周期的影响） |

👉 这些工具属于“信息因子系”，特别适合强化学习和风险控制。

### 5️⃣ **概率与随机过程类**

这类方法让系统更贴近“自然市场生成机制”，而不是仅仅做经验过滤。

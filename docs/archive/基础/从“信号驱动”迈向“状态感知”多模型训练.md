## 🧱 二、什么是多目标学习（Multi-Task Learning）？

### 定义：
让一个模型同时学习 **多个相关任务**，共享底层特征，提升泛化能力。

### 类比：
就像一个交易员：
- 看价格 → 判断“方向”
- 看 K 线影线 → 判断“波动是否剧烈”
- 看连续阳线 → 判断“趋势能走多远”

他不是只猜“涨跌”，而是在**综合判断市场状态**。

我们要让 LightGBM 也学会这种“综合判断”。

---

## 📌 三、标签从哪来？——你已经有数据！

假设你有价格序列：`close[0], close[1], ..., close[t]`

你可以用未来 N 根 K 线的数据构造多个标签：

### ✅ 目标 1：方向（分类任务）
```python
# 未来 5 根 K 线的总收益率
future_return_5 = (close[t+5] - close[t]) / close[t]

# 标签：涨 or 跌
y_direction = 1 if future_return_5 > 0 else 0
```

👉 这是你已经在做的“涨跌预测”。

---

### ✅ 目标 2：波动率（回归任务）
```python
# 未来 5 根 K 线的绝对波动
y_volatility = abs(future_return_5)

# 或者用真实波动：未来 5 根 K 线的平均 ATR
y_volatility = np.mean(atr[t+1:t+6])
```

👉 这个标签告诉你：“这次上涨/下跌会有多猛？”

---

### ✅ 目标 3：趋势持续性（回归任务）
```python
def count_consecutive_same_sign(returns, threshold=0.001):
    """计算连续同号收益率的长度"""
    count = 0
    current_sign = None
    for r in returns:
        if abs(r) < threshold:  # 忽略太小的波动
            continue
        if np.sign(r) == current_sign:
            count += 1
        else:
            current_sign = np.sign(r)
            count = 1
    return count

# 计算未来 10 根 K 线的趋势持续性
future_returns = [(close[i+1] - close[i]) / close[i] for i in range(t+1, t+11)]
y_trend_duration = count_consecutive_same_sign(future_returns)
```

👉 这个标签告诉你：“如果涨了，能持续多久？”

## 🛠 四、如何训练？——两种方案

### ✅ 方案 1：多个单目标模型（简单，推荐初学者）

分别训练 3 个模型：

```python
# 模型 1：预测方向（分类）
model_dir = lgb.LGBMClassifier()
model_dir.fit(X_train, y_direction)

# 模型 2：预测波动率（回归）
model_vol = lgb.LGBMRegressor()
model_vol.fit(X_train, y_volatility)

# 模型 3：预测趋势持续性（回归）
model_dur = lgb.LGBMRegressor()
model_dur.fit(X_train, y_trend_duration)
```

在预测时：

```python
pred_dir = model_dir.predict_proba(X_test)[:, 1]  # P(上涨)
pred_vol = model_vol.predict(X_test)
pred_dur = model_dur.predict(X_test)

# 综合决策
if pred_dir > 0.6 and pred_vol > 0.02 and pred_dur > 3:
    position_size = base_size * 1.5
else:
    position_size = base_size * 0.5
```

✅ **优点**：简单、稳定、可解释性强  
✅ **适合你当前阶段**

---

### ✅ 方案 2：真正的“多输出”模型（进阶）

LightGBM 本身不支持多输出，但你可以：

#### 方法 A：用 `MultiOutputRegressor` 包装

```python
from sklearn.multioutput import MultiOutputRegressor

# 合并多个目标
Y = np.column_stack([y_volatility, y_trend_duration])  # 注意：方向是分类，不能一起训

# 使用回归任务
model_multi = MultiOutputRegressor(lgb.LGBMRegressor())
model_multi.fit(X_train, Y)

# 预测
pred_vol, pred_dur = model_multi.predict(X_test)[0]
```

#### 方法 B：自定义损失函数（高级）

你可以写一个**组合损失函数**，让模型同时优化方向和波动：

```python
def multi_task_objective(y_true_dir, y_pred_dir, y_true_vol, y_pred_vol):
    # 二分类交叉熵 + MSE
    loss_dir = log_loss(y_true_dir, y_pred_dir)
    loss_vol = mean_squared_error(y_true_vol, y_pred_vol)
    return 0.7 * loss_dir + 0.3 * loss_vol
```

⚠️ 这需要自己实现训练循环，适合有 PyTorch/TensorFlow 经验的人。

## 🧩 六、实战标签构造示例（完整代码框架）

```python
import numpy as np
import pandas as pd
from sklearn.model_selection import TimeSeriesSplit

def create_multi_target_labels(close, n_forward=5, vol_window=5):
    """
    构造多目标标签
    """
    labels = {
        'direction': [],
        'volatility': [],
        'trend_duration': []
    }
    
    for i in range(len(close) - n_forward - 10):  # 预留空间
        # 目标 1：方向
        ret_5 = (close[i + n_forward] - close[i]) / close[i]
        labels['direction'].append(1 if ret_5 > 0 else 0)
        
        # 目标 2：波动率
        vol = abs(ret_5)
        labels['volatility'].append(vol)
        
        # 目标 3：趋势持续性
        future_rets = np.diff(close[i+1:i+11]) / close[i+1:i+10]
        dur = 0
        sign = None
        for r in future_rets:
            if abs(r) < 1e-4:
                continue
            if np.sign(r) == sign:
                dur += 1
            else:
                sign = np.sign(r)
                dur = 1
        labels['trend_duration'].append(dur)
    
    return pd.DataFrame(labels)

# 使用
```

## ✅ 总结：你不需要“新标签”，而是“新视角”

| 任务 | 标签来源 | 用途 |
|------|----------|------|
| **方向预测** | 未来涨跌 | 开仓方向 |
| **波动率预测** | 未来绝对收益 | 控制仓位、止损 |
| **趋势持续性** | 连续同向K线数 | 决定是否加仓、持有 |

> 🔥 **你不是缺标签，而是没意识到：未来数据可以生成多个“有用”的学习目标。**

# 深度学习序列特征提取系统

**创建日期**: 2025-10-22  
**版本**: 1.0

## 系统概览

本系统实现了完整的深度学习序列特征提取pipeline，用于从OHLCV时间序列中提取高维特征，然后输入LightGBM进行预测。

### 架构图

```
原始数据 (OHLCV)
    ↓
[订单流特征] → CVD滚动窗口、Taker Buy Ratio等
    ↓
[深度学习序列特征] → Mamba/Transformer (FP16) → 64维向量
    ↓
[传统特征工程] → 小波、Hurst、Hilbert、Spectral等
    ↓
[特征管理] → 动态特征选择（解决新特征无法纳入问题）
    ↓
[降维] → Incremental PCA (64维)
    ↓
[LightGBM] → Warm Start训练 (GPU加速)
    ↓
[交易信号] → 回测评估
```

## 核心组件

### 1. 深度学习序列特征提取器

**文件**: `src/ml_trading/data_tools/dl_sequence_features.py`

#### 支持的后端

| 后端 | 特点 | 复杂度 | 推荐场景 |
|------|------|--------|----------|
| **Mamba** | O(n)线性复杂度，显存友好 | O(n) | 长序列（>200 bars） |
| **FlashAttention** | 2-4x加速，Tensor Core优化 | O(n²) | 中等序列（60-200 bars） |
| **Transformer** | 标准实现，稳定fallback | O(n²) | 任何场景 |

#### 动态特征选择

解决"新特征无法纳入"的问题：

```python
def select_features(self, threshold=0.8, min_features=50, max_features=150):
    """
    动态特征选择：
    1. 按重要性排序
    2. 累积重要性达到阈值
    3. 限制在min_features和max_features之间
    """
```

**流程**:
1. 所有新特征初始重要性为0
2. 训练后更新重要性（指数平滑）
3. 定期衰减（避免早期特征永久主导）
4. 累积阈值选择（默认80%）

### Warm Start训练

```python
def train_with_warm_start(X_train, y_train, prev_model=None, num_boost_round=50):
    """
    使用上一个模型作为初始化，保留旧知识
    """
    if prev_model is not None:
        model = lgb.train(
            params,
            train_data,
            init_model=prev_model,      # 关键：使用之前的模型
            num_boost_round=num_boost_round,
            keep_training_booster=True
        )
    else:
        model = lgb.train(params, train_data, num_boost_round=num_boost_round)
    
    return model
```

## 常见问题

### Q: Mamba安装失败怎么办？

A: 没关系！系统会自动回退到Transformer。Mamba主要用于超长序列（>500 bars），对于120 bars的场景，Transformer已经足够。

### Q: FlashAttention编译失败？

A: 正常现象。FlashAttention需要CUDA编译环境。系统会自动使用标准Transformer，性能略慢但功能完全相同。

### Q: 如何选择seq_length？

A: 
- 短期策略（日内）: 60-120 bars (5-10小时 for 5min)
- 中期策略（swing）: 200-500 bars (17-42小时 for 5min)
- 长期策略（trend）: 500+ bars (使用Mamba)

### Q: 特征数量太多怎么办？

A: 系统有3层降维：
1. 动态特征选择（FeatureManager）：保留80%重要性
2. PCA降维：降到64维
3. LightGBM内部：自动特征选择

### Q: 训练时间多久？

A: RTX 3080上的估算：
- DL特征提取：~5-10 min/month
- 传统特征工程：~2-3 min/month
- LightGBM训练：~1-2 min/month
- **总计**：~10-15 min/month × 6 months = **1-1.5小时**

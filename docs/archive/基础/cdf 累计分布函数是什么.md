`CDF` 是 **Cumulative Distribution Function** 的缩写，中文叫：

> **累积分布函数**

它是概率论和统计学中的一个核心概念，用来描述一个随机变量 **小于或等于某个值的概率**。

---

## 📊 直观理解

假设你有一组数据，比如：
- 某股票的日收益率：`[-2.1%, -1.5%, -0.3%, 0.1%, 0.8%, 1.2%, 3.0%, ...]`

你想知道：
> “收益率 ≤ 1.0% 的概率是多少？”

这就是 `CDF` 要回答的问题。

---

## ✅ CDF 的定义

对于一个随机变量 \( X \)，它的 CDF 定义为：

\[
F(x) = P(X \leq x)
\]

读作：
> “X 小于或等于 x 的概率”

---

## 📈 举个例子

假设你的数据是：`[1, 2, 3, 4, 5]`

| x | CDF(x) = P(X ≤ x) | 说明 |
|---|-------------------|------|
| 0 | 0.0 | 没有数 ≤ 0 |
| 1 | 0.2 | 只有 1 ≤ 1 → 1/5 = 20% |
| 2 | 0.4 | 1,2 ≤ 2 → 2/5 = 40% |
| 3 | 0.6 | 1,2,3 ≤ 3 → 60% |
| 4 | 0.8 | 80% |
| 5 | 1.0 | 100% |
| 6 | 1.0 | 所有数都 ≤ 6 |

这个函数从 0 一路“阶梯式”上升到 1，就是 **累积** 的含义。

---

## 📊 图形表示

```
F(x)
1.0 |         ────────●
0.8 |       ────────●
0.6 |     ────────●
0.4 |   ────────●
0.2 | ────────●
0.0 ●──────────────────► x
    1   2   3   4   5
```

这就是 CDF 的典型形状（离散数据）。

---

## 🔍 在 `TDigest` 中的用途

你在用：

```python
digest.cdf(x)  # 返回 P(X <= x)
```

这在量化交易中非常有用，例如：

### ✅ 场景1：判断当前波动率是否“异常高”
```python
current_atr = 2.5
prob = digest.cdf(2.5)  # 历史中 ATR ≤ 2.5 的比例

if prob > 0.95:
    print("当前波动率处于历史95%高位，可能过热")
```

### ✅ 场景2：归一化特征
```python
# 把原始值映射到 [0,1] 区间，作为模型输入
normalized_vol = digest.cdf(current_volume)
```

这比简单的 `(x - min)/(max - min)` 更鲁棒。

---

## 🔁 和 `quantile` 的关系：互为反函数！

- `quantile(q)`：给我一个概率 `q`，返回对应的值 `x`
- `cdf(x)`：给我一个值 `x`，返回对应的概率 `q`

👉 就像加法和减法、乘法和除法一样，是“逆运算”。

```python
x = digest.quantile(0.95)
p = digest.cdf(x)
# p ≈ 0.95
```

---

## ✅ 总结

| 函数 | 输入 | 输出 | 用途 |
|------|------|------|------|
| `cdf(x)` | 一个数值（如 2.5） | 概率 `[0,1]` | “这个值在历史中排多少？” |
| `quantile(q)` | 概率 `[0,1]`（如 0.95） | 一个数值 | “历史前95%的阈值是多少？” |

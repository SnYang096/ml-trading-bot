非常好的问题，这个其实是**多时间尺度模型设计（multi-timeframe modeling）**的核心。
答案是：✅ **可以一起训练，但要注意分层处理和特征融合方式**，否则容易「时间尺度冲突」。下面我详细拆解给你看。

## 🧩 一、核心结论：

| 模型结构                                     | 是否推荐    | 说明                                                        |
| ---------------------------------------- | ------- | --------------------------------------------------------- |
| **所有周期特征直接扔进一个 LightGBM 模型**             | ⚠️ 不推荐  | 不同时间尺度的特征存在滞后差异，会让模型混乱（如5m的波动噪音影响1h趋势因子）                  |
| **分周期训练 LightGBM 模型（5m / 15m / 1h 各一套）** | ✅ 推荐    | 模型各自学习对应节奏，再融合信号                                          |
| **多周期特征拼接 + Meta Model（二级融合）**           | 🧠 高阶推荐 | 用一层模型预测中低频信号，一层模型捕捉大周期方向，最后融合成综合置信度（类似 ensemble stacking） |

---

## 🧠 二、时间尺度组合的常见结构：

| 组合方式                    | 实现方式                                             | 优点      | 缺点              |
| ----------------------- | ------------------------------------------------ | ------- | --------------- |
| **单模型多尺度特征**            | 直接加入 5m, 15m, 1h 特征字段，例如 `bb_width_5m`, `roc_1h` | 简单      | 模型可能“混乱”，对齐问题复杂 |
| **多模型融合 (Ensemble)**    | 三个 LightGBM 各自训练，输出置信度；再用 meta-model 加权融合        | 稳定泛化好   | 训练复杂            |
| **分周期投票 (Voting)**      | 各周期模型独立发信号，机器人执行“共识交易”                           | 简单稳定    | 反应稍慢            |
| **层次强化 (Hierarchical)** | 大周期信号 → 过滤中周期信号 → 触发小周期执行                        | 最贴合实盘逻辑 | 需策略引擎支持层次信号     |

## 📊 三、周期融合的推荐方案（实盘适用）

| 模块                | 周期      | 模型                  | 功能               | 输出                      |

最终综合信号：

## 🧩 范式详解

### 🚫 1. 直接拼接（不推荐）

```python
# 错误示例：raw 拼接
features_1h = calc_features(df_1h)
features_4h = calc_features(df_4h)
features_1d = calc_features(df_1d)

# 直接 concat → 问题重重

```

**问题：**
- ⚠️ **时间尺度不一致**：1h 的 ROC 与 1d 的 ROC 量级完全不同
- ⚠️ **更新频率不同**：1d 特征每24小时才变一次，模型会“卡住”
- ⚠️ **共线性高**：1h 趋势 vs 4h 趋势，本质是同一信号的延迟版本
- ⚠️ **归一化失效**：用全局 zscore 会引入未来信息

### ✅ 2. 独立建模（Separate Models）—— 适合实盘分层

```python

model_4h = LightGBM().fit(X_4h, y_4h)
model_1d = LightGBM().fit(X_1d, y_1d)

          0.3 * model_4h.predict() + 
          0.2 * model_1d.predict())
```

**优点：**
- ✅ 逻辑清晰，易于调试
- ✅ 可针对不同周期调参（如 BTC 1h 波动大，4h 趋势强）
- ✅ 部署灵活，可动态开关某个周期

**缺点：**
- ❌ 错过“1h 突破 + 4h 压缩 + 1d 支撑”这类**跨周期共振信号**

#### 实现方法：

##### ✅ 方法 1：**跨周期差分/比率编码**

- **TabNet**：自带特征选择，可自动学习周期重要性

## 🧩 2. 多周期数据对齐函数

```python
def align_multi_timeframe_data(df: pd.DataFrame) -> Dict[str, pd.DataFrame]:
    """
    输入原始 tick/1m 数据，输出对齐的 1h, 4h, 1d 数据
    df: index=datetime, columns=['open','high','low','close','volume']
    """
    # 原始高频数据（如1m）
    df_1m = df.copy()

    # 重采样到各周期
    def resample_ohlc(df, freq):
        return df.resample(freq).agg({
            'open': 'first',
            'high': 'max',
            'low': 'min',
            'close': 'last',
            'volume': 'sum'
        }).dropna()

    df_1h = resample_ohlc(df_1m, '1H')
    df_4h = resample_ohlc(df_1m, '4H')
    df_1d = resample_ohlc(df_1m, '1D')

    # 以 1h 为主频率，向前填充大周期数据
    df_4h_aligned = df_4h.reindex(df_1h.index, method='ffill')
    df_1d_aligned = df_1d.reindex(df_1h.index, method='ffill')

    return {
        '1h': df_1h,
        '4h': df_4h_aligned,
        '1d': df_1d_aligned
    }
```

```python
    return (x - x.rolling(window).mean()) / (x.rolling(window).std() + 1e-8)

def rolling_percentile(x, window=100, per=50):
    return x.rolling(window).apply(lambda win: (win <= win[-1]).mean(), raw=True)

def tanh_norm(x):
    return np.tanh(x)

def clip_norm(x, a=0, b=1):
```

## 🌐 4. 核心特征计算函数（Yin-50+ 多周期融合版）

```python
def compute_yin50_multi_tf(df_dict: Dict[str, pd.DataFrame], window=20) -> pd.DataFrame:
    """
    计算 Yin-50+ 多周期融合特征
    输入: {'1h': df, '4h': df, '1d': df}
    输出: 特征 DataFrame (index 同 1h)
    """
    df_1h = df_dict['1h']
    df_4h = df_dict['4h']
    df_1d = df_dict['1d']

    features = {}

    # 辅助变量
    close = df_1h['close']
    volume = df_1h['volume']
    high = df_1h['high']
    low = df_1h['low']
    hl = high - low
    atr_1h = ta.volatility.AverageTrueRange(high, low, close).average_true_range()
    atr_4h = ta.volatility.AverageTrueRange(df_4h['high'], df_4h['low'], df_4h['close']).average_true_range()
    atr_1d = ta.volatility.AverageTrueRange(df_1d['high'], df_1d['low'], df_1d['close']).average_true_range()

    # VWAP
    vwap_1h = ta.volume.VolumeWeightedAveragePrice(
        df_1h['high'], df_1h['low'], df_1h['close'], df_1h['volume']
    ).volume_weighted_average_price()

    # ROC
    roc_1h = ta.momentum.ROCIndicator(close, window=5).roc()
    roc_4h = ta.momentum.ROCIndicator(df_4h['close'], window=3).roc()
    roc_1d = ta.momentum.ROCIndicator(df_1d['close'], window=2).roc()

    # CVD
    cvd = (close - low - (high - close)).cumsum()  # 简化 CVD
    cvd_mom = cvd.diff(3)

    # Swing High/Low (简化)
    swing_high = high.rolling(20).max().shift(1)
    swing_low = low.rolling(20).min().shift(1)

    # === 结构类 ===
    features['S1_dist_to_swing_high'] = (swing_high - close) / (atr_1h + 1e-8)
    features['S2_dist_to_swing_low'] = (close - swing_low) / (atr_1h + 1e-8)
    features['S3_poc_deviation'] = (close - vwap_1h) / (atr_1h + 1e-8)
    features['S4_risk_reward_ratio'] = features['S2_dist_to_swing_low'] / \
        (features['S1_dist_to_swing_high'] + features['S2_dist_to_swing_low'] + 1e-8)

    # === 波动率类 ===
    bb = ta.volatility.BollingerBands(close)
    kc = ta.volatility.KeltnerChannel(high, low, close)
    bb_width = bb.bollinger_width()
    features['V1_bb_atr_ratio'] = bb_width / (atr_1h + 1e-8)
    features['V2_range_ratio'] = hl / (hl.rolling(20).mean() + 1e-8)
    features['V3_atr_regime'] = rolling_percentile(atr_1h, 100)
    features['V4_vol_trend'] = atr_1h.ewm(span=10).mean() / atr_1h.ewm(span=50).mean()
    features['V5_squeeze'] = (bb.band_width() < kc.keltner_width()).astype(int)

    # === 成交量与资金流 ===
    vol_avg = volume.rolling(20).mean()
    features['F1_volume_anomaly'] = volume / (vol_avg + 1e-8)
    features['F2_cvd_momentum'] = cvd_mom / (atr_1h + 1e-8)
    features['F3_updown_vol'] = (volume.where(close > close.shift(1), 0).rolling(5).sum() + 1e-8) / \
                                (volume.where(close < close.shift(1), 0).rolling(5).sum() + 1e-8)
    features['F4_volume_pressure'] = ((close - open) / (high - low + 1e-8)) * volume

    # === 动量与趋势 ===
    hma = ta.trend.HullMovingAverage(close, window=9).hma()
    features['M1_roc_atr'] = roc_1h / (atr_1h + 1e-8)
    features['M2_acceleration'] = roc_1h.diff(1)
    features['M3_hma_slope'] = (hma - hma.shift(1)) / (atr_1h + 1e-8)
    features['M4_trend_consistency'] = (np.sign(roc_1h) == np.sign(roc_4h.reindex(close.index, method='ffill'))).astype(float)
    features['M5_detrended_mom'] = rolling_zscore(roc_1h - roc_1h.rolling(20).mean())

    # === 时间类 ===
    hour = close.index.hour
    day = close.index.dayofweek
    features['T1_hour_sin'] = np.sin(2 * np.pi * hour / 24)
    features['T2_hour_cos'] = np.cos(2 * np.pi * hour / 24)
    features['T3_is_us_open'] = ((hour == 13) & (close.index.minute == 55)).astype(int)
    features['T4_day_sin'] = np.sin(2 * np.pi * day / 7)

    # === 融合 / Meta 特征 ===
    features['X1_compression_energy'] = (1 / (features['V1_bb_atr_ratio'] + 1e-8)) * features['F1_volume_anomaly']
    features['X2_structure_tension'] = (features['S1_dist_to_swing_high'] + features['S2_dist_to_swing_low']) / \
                                       (features['V1_bb_atr_ratio'] + 1e-8)
    features['X3_vwap_breakout'] = np.sign(close - vwap_1h) * features['M1_roc_atr']
    features['X4_trend_vol_align'] = np.sign(roc_1h) * features['V3_atr_regime']
    features['X5_breakout_quality'] = features['V3_atr_regime'] * features['M1_roc_atr']

    # === 高阶统计 ===
    returns = close.pct_change()
    features['H1_price_entropy'] = returns.rolling(10).apply(
        lambda x: shannon_entropy(pd.cut(x, bins=5).codes.value_counts(normalize=True) + 1e-8), raw=False
    )
    coeffs = cwt(returns.fillna(0).values, ricker, np.arange(1, 6))
    features['H2_wavelet_lowfreq'] = np.mean(coeffs[0:2]**2, axis=0)
    features['H3_wavelet_highfreq'] = np.mean(coeffs[3:5]**2, axis=0)

    # === 跨周期融合 ===
    features['XF1_trend_1h_vs_4h'] = roc_1h / (roc_4h.reindex(close.index, method='ffill') + 1e-8)
    features['XF2_vol_regime_4h_vs_1d'] = features['V3_atr_regime'] - rolling_percentile(atr_1d.reindex(close.index, method='ffill'), 100)
    features['XF3_compression_4h'] = (bb_width / (atr_4h.reindex(close.index, method='ffill') + 1e-8)).rolling(4).mean()

    # === 轻量辅助 ===
    features['A1_session_return'] = (close - open) / (open + 1e-8)
    features['A2_is_inside_bar'] = (high <= high.shift(1)) & (low >= low.shift(1))

    # 归一化（示例）
    for col in features:
        if col.startswith('V') or col.startswith('M') or col.startswith('F'):
            features[col] = rolling_zscore(pd.Series(features[col]), window=20)

    return pd.DataFrame(features, index=close.index)
```

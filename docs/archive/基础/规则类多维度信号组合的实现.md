# 规则类多维度信号组合的实现

本文档详细说明了在交易策略中实现的六种有意义的信号组合，这些组合基于市场行为逻辑，将技术指标和订单流特征有机结合，形成高概率、低噪声的开仓信号。

## 🎯 设计目标

将复杂的特征按市场行为逻辑（蓄势 → 静默 → 突破 → 动能确认）进行组合，形成可解释性强、统计独立性好、风险收益比高的开仓信号组合。

## 📌 信号组合实现详情

### 🔹 1. Spring Load（弹簧加载）

**核心逻辑**：波动率 + 成交量双重压缩，结构紧致，等待爆发
**适用场景**：趋势延续或反转前的极致压缩区

#### ✅ 实现要素：
- ATR压缩分位数 < 0.2
- 成交量压缩分位数 < 0.2
- 结构性压缩（实体比例 < 0.3，ATR比率 < 0.4）
- 压缩置信度 > 0.7

#### 🧩 加分项：
- 持续时间奖励 > 0.1（持续时间长）
- 内部价格密度 > 0.8（内部价格密集）

#### 💬 命名由来：
像弹簧被压到极限，随时可能反弹。适用于震荡末期。

### 🔹 2. Silent Accumulation（静默吸筹）

**核心逻辑**：突破前的低波动静默 + 内部价格密集 + 订单吸收迹象
**适用场景**：主力吸筹、准备向上突破

#### ✅ 实现要素：
- 突破前静默ATR分位数 < 0.15
- 内部价格密度 > 0.75
- 订单吸收迹象（小实体大成交量）
- 方向有序（熵 < 0.3）

#### 🧩 加分项：
- 弱CVD动量存在但未爆发（暗中积累）
- 高流动性时间窗口

#### 💬 命名由来：
市场表面平静，但内部有资金悄悄建仓，是"聪明钱"行为。

### 🔹 3. Breakout Ignition（突破点火）

**核心逻辑**：压缩后首次爆发 + 成交量spike + CVD动能确认
**适用场景**：趋势启动的第一根强势K线

#### ✅ 实现要素：
- 首次突破K线
- 价格突破强度 >= 2.0
- 成交量spike >= 2.0
- 扩张置信度 > 0.7

#### 🧩 加分项：
- 量价CVD三者共振
- 向上突破价值区
- 大单主导

#### 💬 命名由来：
"点火"意味着趋势正式启动，是最佳入场时机之一。

### 🔹 4. ATR Squeeze + POC Escape（ATR压缩 + POC逃逸）

**核心逻辑**：波动率压缩 + 突破关键价值节点（POC）+ 动量跟进
**适用场景**：区间突破、价值区逃离

#### ✅ 实现要素：
- ATR压缩权重激活
- POC方向突破
- 动量收敛触发
- 价格突破强度 >= 1.5

#### 🧩 加分项：
- 贴近POC边缘突破
- 强劲CVD动量

#### 💬 命名由来：
结合经典ATR Squeeze与订单流POC理论，兼具技术与微观结构逻辑。

### 🔹 5. Volatility Vacuum（波动真空）

**核心逻辑**：极低波动 + 极低成交量 + 高价格密度 → 预示剧烈波动即将来临
**适用场景**：盘整末端，适合挂单等待突破

#### ✅ 实现要素：
- 波动密集度 > 0.8（波动密集但幅度小）
- 成交量压缩分位数 < 0.1
- 突破前静默ATR分位数 < 0.1
- 内部价格密度 > 0.9

#### 🧩 加分项：
- 方向即将成型（熵 < 0.2）
- 高流动性时间窗口

#### 💬 命名由来：
市场像"真空"一样缺乏波动，一旦打破，能量巨大。

### 🔹 6. Momentum Fusion（动量融合）

**核心逻辑**：多种动量信号汇聚 + 压缩释放 + CVD确认
**适用场景**：强势趋势中的二次启动

#### ✅ 实现要素：
- 动量收敛触发
- 强CVD动量 = True
- 成交量spike >= 1.5
- 扩张置信度 > 0.6

#### 🧩 加分项：
- POC方向与趋势一致
- 前期压缩时间长

#### 💬 命名由来：
多种动量因子"融合"在一起，形成强大推力。

## 🧩 使用建议

| 用途 | 方法 |
|------|------|
| **策略分类** | 将不同信号分组，分别测试胜率、盈亏比 |
| **动态仓位** | 高置信信号（如Ignition）加大仓位 |
| **过滤机制** | 只在Spring Load/Vacuum后允许开仓 |
| **多时间框架** | 大周期Spring Load，小周期Breakout Ignition入场 |

## ✅ 信号命名与逻辑对照表

| 名称 | 核心特征 | 适合方向 | 信号类型 |
|------|---------|----------|----------|
| **Spring Load** | ATR + Volume压缩 + 结构紧致 | 多/空 | 蓄势 |
| **Silent Accumulation** | 静默 + 吸筹 + 价格密集 | 多（为主）| 吸筹 |
| **Breakout Ignition** | 首根突破 + 量能爆发 + CVD共振 | 多/空 | 启动 |
| **ATR Squeeze + POC Escape** | ATR压缩 + POC突破 + 动量 | 多/空 | 突破 |
| **Volatility Vacuum** | 极低波动 + 高密度 | 多/空 | 预警 |
| **Momentum Fusion** | 多因子动量共振 | 趋势方向 | 延续 |

## 📊 代码实现

所有信号组合已在`strategy.py`文件中实现，通过`_check_simplified_entry_conditions`方法按优先级顺序检查各个信号条件。每个信号都有独立的检测方法，便于单独测试和优化。

## 🎯 总结

这六种信号组合提供了从不同角度识别高概率交易机会的方法，每种信号都有明确的市场含义和适用场景，使得策略更加透明、可解释和易于优化。

## 🔬 Spectral修复速查

### 问题

```
所有15个Spectral特征重要性=0
原因：标量赋值，所有行相同，无信息量
```

### 修复

```python
# 错误：
spectral = calculate_once(entire_data)
df['spectral'] = spectral  # 标量

# 正确：
for i in range(n):
    spectral[i] = calculate(data[i-100:i])
df['spectral'] = spectral  # 时间序列
```

### 结果

```
修复前：0.0
修复后：1072.2
成功率：100% (15/15)
```

# 统一回测引擎改造方案

## 📋 当前问题分析

### 1. 代码重复情况

| 位置 | 回测函数 | 功能 | 复杂度 |
|------|---------|------|--------|
| `rolling.py` | `evaluate_signal_performance` | 简单向量化回测 | ⭐ 低 |
| `ts-dim-compare` | `calculate_strategy_returns_from_predictions` | 简单向量化回测 | ⭐ 低 |
| `vectorbot.py` | `VectorBotBacktest` | 完整回测系统（止损/止盈/仓位管理） | ⭐⭐⭐ 高 |

### 2. 重复的代码逻辑

**共同功能**:
- ✅ 计算累计收益
- ✅ 计算胜率
- ✅ 计算盈亏比
- ✅ 计算最大回撤
- ✅ 计算夏普比率

**差异**:
- `evaluate_signal_performance`: 使用 `signal_strength` 作为仓位
- `calculate_strategy_returns_from_predictions`: 固定 100% 仓位
- `VectorBotBacktest`: 动态仓位 + 止损/止盈 + 多持仓管理

---

## 🎯 改造目标

1. **统一接口**: 所有回测都使用同一个引擎
2. **灵活配置**: 支持简单回测和完整回测两种模式
3. **向后兼容**: 不破坏现有代码
4. **性能优化**: 简单回测使用向量化，完整回测使用逐笔模拟

---

        """

def unified_backtest(
    signals_df: Optional[pd.DataFrame] = None,
    predictions: Optional[np.ndarray] = None,
    price_data: Optional[pd.DataFrame] = None,
    future_returns: Optional[pd.Series] = None,
    mode: str = "simple",  # "simple" 或 "full"
    **kwargs

    """
    统一的回测函数。

        signals_df: 信号 DataFrame（包含 signal_strength, signal 等列）

    Returns:
        回测结果字典
    """
    if mode == "simple":
        return _simple_backtest(signals_df, predictions, price_data, future_returns, **kwargs)
    else:
        return _full_backtest(signals_df, predictions, price_data, **kwargs)

### 实施步骤

#### 步骤 1: 添加新的初始化方法

```python
def __init__(
    self,
    model_path: Optional[str] = None,
    signals_df: Optional[pd.DataFrame] = None,
    predictions: Optional[np.ndarray] = None,
    price_data: Optional[pd.DataFrame] = None,
    future_returns: Optional[pd.Series] = None,
    mode: str = "full",
    symbol: Optional[str] = None,
    initial_capital: float = 100000,
    **kwargs
):
    """统一的初始化方法。"""
    self.mode = mode
    self.symbol = symbol or "UNKNOWN"
    self.initial_capital = initial_capital
    self.capital = initial_capital
    
    # 根据输入方式初始化
    if model_path:
        self.model_path = model_path
        self.load_model()
        self.is_rolling_format = hasattr(self, 'cls_pipeline')
    elif signals_df is not None:
        self._init_from_signals(signals_df, future_returns)
    elif predictions is not None and price_data is not None:
        self._init_from_predictions(predictions, price_data)
    else:
        raise ValueError("必须提供 model_path、signals_df 或 (predictions, price_data)")
```

#### 步骤 2: 添加简单回测模式

```python
def _run_simple_backtest(self) -> Dict[str, Any]:
    """简单向量化回测（用于 rolling 和 ts-dim-compare）。"""
    if hasattr(self, 'signals_df') and hasattr(self, 'future_returns'):
        # 使用 signals_df 和 future_returns
        return self._evaluate_signals_simple()
    elif hasattr(self, 'predictions') and hasattr(self, 'price_data'):
        # 使用 predictions 和 price_data
        return self._evaluate_predictions_simple()
    else:
        raise ValueError("缺少必要的回测数据")
```

## ⚠️ 注意事项

### 1. 向后兼容

- 保留 `evaluate_signal_performance` 函数（标记为 deprecated）
- 保留 `backtest_classification_model` 函数（标记为 deprecated）
- 逐步迁移现有代码

### 2. 性能考虑

- **简单模式**: 使用向量化操作，性能与当前实现相同
- **完整模式**: 使用逐笔模拟，性能较慢但更真实

### 3. 接口统一

- 所有回测函数返回相同格式的结果字典
- 包含：`total_trades`, `total_return`, `win_rate`, `profit_factor`, `max_drawdown` 等

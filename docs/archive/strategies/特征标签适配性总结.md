## 二、各策略详细适配性分析

### 2.1 SR 反转策略

#### 标签设计 ✅
- **类型**：二元标签（≥2R 成功率）
- **定义**：在 SR 区入场后，50根K线内是否先触达 +2R 止盈 而非 -1R 止损？
- **适配性**：✅ **完美匹配** - 直接优化策略核心目标（高 R/R 成功率）

#### 特征设计 ✅
- **频率域**：WPT 多尺度分解（识别不同周期的 SR）
- **时间域**：Hurst（趋势持续性）+ Hilbert（CVD 相位领先）
- **空间域**：SR 强度（SQS、距离、密度）+ VPVR（PVP、HVN）+ ZigZag（结构锚点）

#### 特征-标签适配性 ✅
- **特征关注点**：SR 强度 + 流动性验证 + 波动状态
- **标签目标**：是否值得冒 1R 风险博 2R 收益
- **适配性**：✅ **高度匹配**
  - SR 强度 → 判断"在哪里交易"（强共识区）
  - 流动性验证 → 判断"是否有主力参与"（CVD 相位领先）
  - 波动状态 → 判断"是否适合反转"（避免高波动假信号）

---

### 2.2 SR 突破策略

#### 标签设计 ✅
- **类型**：连续标签（实现 R/R）
- **定义**：突破后最大有利偏移 / 最大不利偏移（MFE/MAE），截断在 [0, 3] 区间
- **适配性**：✅ **完美匹配** - 区分"真突破跑远" vs "假突破回抽"

#### 特征设计 ✅
- **频率域**：WPT 多尺度能量比 + Spectrum 主频迁移
- **时间域**：Hurst（趋势持续性）+ Hilbert（CVD 相位领先）
- **空间域**：突破质量（成交量、VPER）+ 真空区识别（VPVR LVN）+ ZigZag（突破点）

#### 特征-标签适配性 ✅
- **特征关注点**：突破质量 + 动能持续性 + 真空区识别
- **标签目标**：预期盈利质量（R/R）
- **适配性**：✅ **高度匹配**
  - 突破质量 → 判断"是否真实"（成交量、VPER、站稳比例）
  - 动能持续性 → 判断"能否延续"（WPT 能量比、Hurst、ROC）
  - 真空区识别 → 判断"是否有跑道"（VPVR LVN、Spectrum 迁移）

---

### 2.3 压缩区突破策略

#### 标签设计 ✅
- **类型**：三元标签（方向+质量）
- **定义**：-1（向下有效突破）、0（假突破/回补）、+1（向上有效突破）
- **适配性**：✅ **完美匹配** - 同时捕捉方向与突破质量

#### 特征设计 ✅
- **频率域**：WPT 高频能量 + Spectrum 频谱平坦度
- **时间域**：突破触发（range 扩张、Volume spike、CVD 突变）
- **空间域**：压缩强度（BB Width、ATR 比值）+ 方向确认（body/range、TBR）+ VPVR（SR 密度）+ ZigZag（压缩区间）

#### 特征-标签适配性 ✅
- **特征关注点**：压缩强度 + 突破触发 + 方向确认
- **标签目标**：突破方向 + 真假判断
- **适配性**：✅ **高度匹配**
  - 压缩强度 → 判断"弹簧压得多紧"（BB Width、ATR、Spectrum 平坦度）
  - 突破触发 → 判断"是否启动"（range 扩张、Volume spike、CVD 突变）
  - 方向确认 → 判断"方向+有效性"（body/range、TBR、WPT 高频能量方向）

---

### 2.4 趋势跟踪策略

#### 标签设计 ✅
- **类型**：百分位标签（Rank）
- **定义**：未来 50 根K线收益率在滚动窗口中的分位数（0~1）
- **适配性**：✅ **完美匹配** - 对波动率/分布鲁棒，适应不同品种

#### 特征设计 ✅
- **频率域**：WPT 低频斜率 + Spectrum 主频周期
- **时间域**：Hurst（趋势持续性）+ Hilbert（频率稳定性）+ ROC（动量方向）
- **空间域**：收益率百分位（Rank 的天然特征）+ VPVR（流动性聚集区）+ ZigZag（趋势结构锚点）

#### 特征-标签适配性 ✅
- **特征关注点**：趋势强度 + 资金验证 + 节奏感知
- **标签目标**：趋势相对强度（Rank）
- **适配性**：✅ **高度匹配**
  - 趋势强度 → 判断"趋势有多强"（Hurst、ADX、WPT 低频斜率）
  - 资金验证 → 判断"主力是否在场"（CVD 趋势、MFE/MAE、Volume 同向率）
  - 节奏感知 → 判断"趋势节奏"（Hilbert 频率稳定性、Spectrum 周期）

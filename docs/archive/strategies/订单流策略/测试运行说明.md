## 运行方式

### 方式 1：直接运行（需要安装依赖）
```bash

python3 tests/test_vpin_features.py
```

```bash
cd /home/yin/trading/ml_trading_bot
pytest tests/test_vpin_features.py -v

```

## 故障排除

### 问题 1：ModuleNotFoundError
**解决方案**：
```bash
# 安装依赖
pip install numpy pandas scipy pytest

# 或使用 requirements.txt
pip install -r requirements.txt
```

### 问题 2：测试失败
**检查**：
1. 确保 tick 数据格式正确（包含 price, volume, side）
2. 确保 side 值为 1 或 -1
3. 检查时间索引是否为 DatetimeIndex

### 问题 3：性能不达标
**可能原因**：
1. 系统资源不足
2. Numba 未安装（MAD 计算会变慢）
3. 数据量过大

**解决方案**：
- 安装 numba：`pip install numba`
- 减少测试数据量
- 检查系统资源

## 🎓 技术总结

### 代码变更

```python
# 修复前（错误）：
spectral_centroid = calculate_on_entire_series(data)  # 标量
df['spectral'] = spectral_centroid  # 所有行相同

# 修复后（正确）：
for i in range(n_samples):
    window_data = data[i-window:i]
    spectral_centroid[i] = calculate(window_data)  # 数组
df['spectral'] = spectral_centroid  # 每行不同
```

### 验证方法

```python
# 检查特征是否为时间序列
unique_count = df['spectral_centroid'].nunique()
```

# 📊 收益率和胜率详细说明

## 🎯 指标定义

### 1. 胜率（Win Rate）
- **定义**：盈利交易次数 / 总交易次数
- **单位**：百分比
- **示例**：89.69% = 100次交易中约90次盈利

### 2. 总收益（Total Return）
- **定义**：所有交易收益率的简单累加
- **单位**：小数（需要×100转为百分比）
- **示例**：0.7695 = 76.95% 累计收益率

---

## 💡 重要说明

### ⚠️ 总收益不是复利

代码中的 `total_return` 是**简单相加**：

```python
# 每次交易的收益率
return_1 = 0.002  # 0.2%
return_2 = 0.003  # 0.3%
...
return_485 = 0.001  # 0.1%

# 总收益（简单加总）
total_return = return_1 + return_2 + ... + return_485 = 0.7695
```

### ✅ 实际交易应该用复利

```python
# 复利计算
final_capital = initial_capital × (1 + r1) × (1 + r2) × ... × (1 + r485)
```

复利效果会比简单加总更显著（如果持续盈利）。


- 需要持续监控实盘表现
- 建议设置止损和风控

### 3. 资金管理

- 建议每次使用10-20%仓位
- 留有足够的风险缓冲

### 💡 为什么这么多？

**堆叠策略**：对多个信号源都做小波变换
```python
# 1. close价格的小波分解 → ~20个特征
close_wavelet_energy, close_wavelet_detail_1, detail_2, ...

# 2. volume的小波分解 → ~20个特征
volume_wavelet_energy, volume_wavelet_detail_1, ...

# 3. CVD的小波分解 → ~20个特征 
cvd_wavelet_energy, cvd_wavelet_detail_1, ...

# 4. taker_buy_ratio的小波分解 → ~20个特征
tbr_wavelet_energy, tbr_wavelet_detail_1, ...

# 总计：80个小波特征！
```

**衍生指标**：创建了大量组合特征
```python
compression_energy = (1 / bb_width) * volume_anomaly
structure_tension = (dist_high + dist_low) / bb_width
cvd_divergence_strength = price_momentum - cvd_slope_normalized
trend_volatility_alignment = sign(roc_5) * atr_percentile
...
```

### 💡 为什么更少？

**精简策略**：只对核心信号做高级变换

```python
# 1. 只对close做WPT → 36个特征
#    - 8个频带 × (能量 + 均值 + 标准差 + 能量比例)
#    - Shannon熵、能量集中度、高低频比等全局特征
wpt_aaa_energy, wpt_aad_energy, ..., wpt_ddd_energy  # 8个频带
wpt_aaa_mean, wpt_aad_mean, ...                       # 8个均值
wpt_aaa_std, wpt_aad_std, ...                         # 8个标准差
wpt_aaa_energy_ratio, ...                             # 8个能量比
wpt_shannon_entropy, wpt_energy_concentration, ...    # 全局特征

# 2. Hurst指数 → 6个特征
hurst                           # 趋势持续性
hurst_deviation                 # 偏离随机游走的程度
hurst_trend_signal             # 是否趋势市
hurst_mean_revert_signal       # 是否均值回归市
hurst_change                    # Hurst变化率
hurst_acceleration              # Hurst加速度

# 3. 基础指标 → ~26个
#    没有对volume/CVD/TBR再做小波
#    只保留必要的技术指标
```

## 🎭 两种设计哲学

### 基线模型：广撒网策略

```
理念：让模型自己学习哪些特征有用
方法：
  - 对每个信号源都做小波变换
  - 创建大量衍生组合特征
  - 时间特征、结构特征全加上
  
优点：✅ 信息丰富，不遗漏任何可能有用的信号
缺点：⚠️ 特征冗余，计算慢，可能过拟合
```

### 增强模型：精准打击策略

```
理念：用最本质的特征捕捉市场规律
方法：
  - WPT替代多个小波（一次性更精细分解）
  - Hurst指数判断市场状态（趋势vs震荡）
  - Shannon熵衡量市场混乱度
  
优点：✅ 简洁高效，理论优雅，计算快
缺点：⚠️ 可能错过某些有用信号
```

## 💡 为什么"特征少"可能是好事？

### 奥卡姆剃刀原理

> "如无必要，勿增实体"  
> 在解释力相当的情况下，越简单的模型越好

### 过拟合风险

```
特征数量 vs 数据量:

基线模型: 142特征 / 8928样本 ≈ 1:63
增强模型: 68特征 / 8928样本 ≈ 1:131

样本/特征比越高 → 过拟合风险越低
```

### 机器学习经验

很多时候：
- 训练集：更多特征 = 更高准确率
- 测试集：精简特征 = 更好泛化

**示例（经典案例）：**
```
模型A: 1000个特征，训练95%，测试75%  ← 过拟合
模型B: 50个特征，训练88%，测试90%   ← 泛化好
```

## 🎉 总结

### 答案：为什么增强模型特征更少？

1. **设计理念不同**
   - 基线：堆叠所有可能有用的特征（广撒网）
   - 增强：只用最本质的高级特征（精准打击）

2. **基线模型有大量小波变换**
   - close + volume + CVD + TBR 各做一次小波
   - 产生80个小波特征
   - 增强模型只对close做WPT（但更精细）

3. **基线模型有大量衍生指标**
   - compression_energy, structure_tension等
   - 增强模型用Hurst和熵替代了这些

### 最重要的教训：

> ✅ **基线模型142特征虽多，但OOS表现优秀（91.24%）**

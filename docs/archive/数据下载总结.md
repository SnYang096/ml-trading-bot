## 🆘 常见问题

### Q1: 下载速度很慢怎么办？

**A**: 
- 检查网络连接
- 关闭其他下载任务
- 尝试其他时段（避开高峰期）
- 使用有线网络代替WiFi

### Q2: 提示404错误？

**A**: 正常现象，可能原因：
- 该币种在该时期未上线
- Binance未提供该月数据
- 脚本会自动跳过，继续下载其他月份

### Q3: 磁盘空间不够？

**A**: 解决方案：
1. 清理不需要的文件
2. 使用更大的硬盘
3. 只下载需要的币种/时间范围
4. 分批下载

### Q4: 如何删除已下载数据？

**A**: 直接删除文件：
```powershell
# 删除所有BTC数据
Remove-Item D:\GitHub\trading\rlbot\data\agg_data\BTCUSDT-*.zip

# 删除2021年的数据
Remove-Item D:\GitHub\trading\rlbot\data\agg_data\*-2021-*.zip

# 删除全部
Remove-Item D:\GitHub\trading\rlbot\data\agg_data\*.zip
```

### Q5: 下载中断后如何继续？

**A**: 直接重新运行脚本：
```powershell
.\download_to_agg_data.ps1
# 选择同样的选项
# 已下载的会自动跳过
```

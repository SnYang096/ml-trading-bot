## 🔥 正确的合并方式(非常重要)

**❌ 错误做法: 功能式合并**
```python
class UnifiedSafety:
    def forward(x):
        ood = self.ood_branch(x)
        surv = self.survival_branch(x)
        return aggregate(ood, surv)
```
→ 这只是把三个模块塞进一个类，**本质没变**

**✅ 正确做法: 职责坍缩**

### 只保留一个 Safety Head，但用多视角特征

```python
class SafetyHead(nn.Module):
    def __init__(self):
        self.net = nn.Sequential(...)
        
    def forward(self, features):
        """
        features 包含：
        - distribution_shift_score  # 以前的 OOD
        - path_hazard_score         # 以前的 Survival  
        - execution_context
        - regime_state
        """
        
        # 单一输出
        p_extinct = self.net(features)
        
        return {
            'extinction_prob': p_extinct,
            'allow_prob': sigmoid(τ - p_extinct),
            'size_cap': exp(-α * p_extinct)
        }
```

---

### 关键区别：

**之前：**
```
OOD → score_ood
Survival → score_surv  
Safety → aggregate(score_ood, score_surv)
```
→ 三次 forward，三个 loss

**现在：**
```
Safety(features_all) → p_extinct
```
→ 一次 forward，一个 loss

# 实时流数据补全机制设计

## 一、问题分析

### 1.1 数据缺失场景

实时流可能因为以下原因导致数据缺失：

1. **网络中断**
   - WebSocket 连接断开
   - 网络波动导致消息丢失
   - 交易所服务器故障

2. **系统重启**
   - 程序崩溃重启
   - 服务器重启
   - 手动重启

3. **延迟接收**
   - WebSocket 消息延迟
   - 交易所推送延迟
   - 系统负载过高

4. **数据质量问题**
   - 收到重复数据
   - 收到乱序数据
   - 收到无效数据

### 1.2 数据缺失的影响

| 影响 | 严重程度 | 说明 |
|------|---------|------|
| **特征计算错误** | ⚠️ 高 | 某些特征需要连续数据，缺失会导致计算错误 |
| **模型预测偏差** | ⚠️ 高 | 特征错误导致模型预测不准确 |
| **交易信号错误** | ⚠️ 极高 | 可能导致错误的交易决策 |
| **历史窗口不完整** | ⚠️ 中 | 影响后续特征计算 |

## 二、数据补全策略

### 2.1 检测数据缺失

#### 方法1：时间戳连续性检查

```python
def detect_missing_bars(
    df: pd.DataFrame,
    expected_interval: pd.Timedelta,
    tolerance: pd.Timedelta = None,
) -> List[pd.Timestamp]:
    """
    检测缺失的 K线数据
    
    Args:
        df: 已有的 K线数据（按时间排序）
        expected_interval: 期望的 K线间隔（如 15分钟）
        tolerance: 允许的时间误差（默认 10%）
    
    Returns:
        缺失的时间戳列表
    """
    if len(df) < 2:
        return []
    
    if tolerance is None:
        tolerance = expected_interval * 0.1
    
    df_sorted = df.sort_values("timestamp")
    missing_timestamps = []
    
    for i in range(len(df_sorted) - 1):
        current_time = df_sorted.iloc[i]["timestamp"]
        next_time = df_sorted.iloc[i + 1]["timestamp"]
        expected_next = current_time + expected_interval
        
        # 检查是否有缺失
        gap = next_time - current_time
        if gap > expected_interval + tolerance:
            # 计算缺失的时间戳
            missing_count = int((gap - tolerance) / expected_interval)
            for j in range(1, missing_count + 1):
                missing_time = current_time + expected_interval * j
                if missing_time < next_time - tolerance:
                    missing_timestamps.append(missing_time)
    
    return missing_timestamps
```

```python
def detect_missing_from_questdb(
    symbol: str,
    timeframe: str,
    start_time: pd.Timestamp,
    end_time: pd.Timestamp,
    questdb_url: str = "http://localhost:9000",
) -> List[pd.Timestamp]:
    """
    通过查询 QuestDB 检测缺失的数据
    
    Args:
        symbol: 交易对符号
        timeframe: 时间框架
        start_time: 开始时间
        end_time: 结束时间
        questdb_url: QuestDB URL
    
    Returns:
        缺失的时间戳列表
    """
    import requests
    
    # 查询 QuestDB 中已有的数据
    query = f"""
    SELECT DISTINCT timestamp
    FROM klines
    WHERE symbol = '{symbol}' 
      AND timeframe = '{timeframe}'
      AND timestamp >= '{start_time}'
      AND timestamp <= '{end_time}'
    ORDER BY timestamp
    """
    
    try:
        response = requests.post(f"{questdb_url}/exec", data=query, timeout=5)
        response.raise_for_status()
        
        import io
        df_db = pd.read_csv(io.StringIO(response.text))
        
        if len(df_db) == 0:
            # 完全没有数据，返回整个时间段
            return pd.date_range(start_time, end_time, freq=timeframe).tolist()
        
        # 生成期望的时间序列
        expected_times = pd.date_range(start_time, end_time, freq=timeframe)
        existing_times = pd.to_datetime(df_db["timestamp"])
        
        # 找出缺失的时间戳
        missing = expected_times.difference(existing_times)
        return missing.tolist()
    
    except Exception as e:
        print(f"⚠️ 查询 QuestDB 失败: {e}")
        return []
```

```python
class DataGapFiller:
    """数据缺失补全器"""
    
    def __init__(self, exchange: Any):
        """
        Args:
            exchange: ccxt Exchange 实例
        """
        self.exchange = exchange
    
    def download_missing_bars(
        self,
        symbol: str,
        missing_timestamps: List[pd.Timestamp],
        timeframe: str,
    ) -> pd.DataFrame:
        """
        下载缺失的 K线数据
        
        Args:
            symbol: 交易对符号（ccxt 格式）
            missing_timestamps: 缺失的时间戳列表
            timeframe: 时间框架（ccxt 格式，如 '15m'）
        
        Returns:
            下载的 K线数据 DataFrame
        """
        if not missing_timestamps:
            return pd.DataFrame()
        
        # 转换时间框架格式
        ccxt_timeframe = self._convert_timeframe(timeframe)
        
        # 按时间段分组下载（避免单次请求过大）
        all_bars = []
        
        # 找到时间范围
        start_time = min(missing_timestamps)
        end_time = max(missing_timestamps)
        
        # 下载数据
        try:
            since = int(start_time.timestamp() * 1000)  # 毫秒时间戳
            limit = len(missing_timestamps) + 100  # 多下载一些，避免边界问题
            
            ohlcv = self.exchange.fetch_ohlcv(
                symbol,
                timeframe=ccxt_timeframe,
                since=since,
                limit=limit,
            )
            
            # 转换为 DataFrame
            df = pd.DataFrame(
                ohlcv,
                columns=["timestamp", "open", "high", "low", "close", "volume"],
            )
            df["timestamp"] = pd.to_datetime(df["timestamp"], unit="ms")
            
            # 过滤出缺失的时间戳
            df = df[df["timestamp"].isin(missing_timestamps)]
            
            return df
        
        except Exception as e:
            print(f"❌ 下载缺失数据失败: {e}")
            return pd.DataFrame()
    
    def _convert_timeframe(self, timeframe: str) -> str:
        """转换时间框架格式（15T -> 15m）"""
        if "T" in timeframe:
            return timeframe.replace("T", "m")
        return timeframe
```

```python
        })
        
        return self.history_df
    
    def _detect_missing_bars(self, new_bar: pd.DataFrame) -> List[pd.Timestamp]:
        """检测缺失的 K线数据"""
        if len(self.history_df) == 0:
            return []
        
        # 计算期望的间隔
        expected_interval = pd.Timedelta(self.timeframe)
        
        # 检查最后一条数据和新数据之间的间隔
        last_timestamp = self.history_df["timestamp"].iloc[-1]
        new_timestamp = new_bar["timestamp"].iloc[0]
        
        gap = new_timestamp - last_timestamp
        
        # 如果间隔大于期望间隔的 1.5 倍，认为有缺失
        if gap > expected_interval * 1.5:
            # 计算缺失的时间戳
            missing_count = int(gap / expected_interval) - 1
            missing_timestamps = [
                last_timestamp + expected_interval * (i + 1)
                for i in range(missing_count)
            ]
            return missing_timestamps
        
        return []
    
    def _fill_missing_bars(self, missing_timestamps: List[pd.Timestamp]):
        """补全缺失的 K线数据"""
        if not self.gap_filler or not missing_timestamps:
            return
        
        # 下载缺失数据
        missing_df = self.gap_filler.download_missing_bars(
            symbol=self._get_ccxt_symbol(),
            missing_timestamps=missing_timestamps,
            timeframe=self.timeframe,
        )
        
        if len(missing_df) == 0:
            print(f"⚠️ 无法下载缺失数据（可能超出交易所历史范围）")
            return
        
        # 按时间顺序插入到历史数据中
        for _, row in missing_df.iterrows():
            # 转换为标准格式
            bar_dict = {
                "timestamp": row["timestamp"],
                "open": row["open"],
                "high": row["high"],
                "low": row["low"],
                "close": row["close"],
                "volume": row["volume"],
            }
            
            # 追加到历史数据
            bar_df = self._bar_to_dataframe(bar_dict)
            self.history_df = pd.concat([self.history_df, bar_df])
            
            # 写入 QuestDB
            self.questdb_writer.write_bar({
                **bar_dict,
                "symbol": self.symbol,
                "timeframe": self.timeframe,
            })
        
        print(f"✅ 已补全 {len(missing_df)} 条缺失数据")
    
    def periodic_gap_check(self):
        """定期检查数据缺失（后台任务）"""
        current_time = time.time()
        
        if current_time - self.last_gap_check < self.gap_detection_interval:
            return
        
        self.last_gap_check = current_time
        
        if not self.auto_fill_gaps or self.history_df is None:
            return
        
        try:
            # 检查最近一段时间的数据完整性
            recent_df = self.history_df.tail(100)  # 检查最近 100 条
            
            if len(recent_df) < 2:
                return
            
            # 检测缺失
            missing = self._detect_missing_in_range(recent_df)
            
            if missing:
                print(f"⚠️ 定期检查发现 {len(missing)} 条缺失数据")
                self._fill_missing_bars(missing)
        
        except Exception as e:
            print(f"⚠️ 定期检查数据缺失时出错: {e}")
    
    def _detect_missing_in_range(self, df: pd.DataFrame) -> List[pd.Timestamp]:
        """在指定范围内检测缺失数据"""
        if len(df) < 2:
            return []
```

```python
class EnhancedRealtimeDataManager(RealtimeDataManager):

    def __init__(
        self,

        timeframe: str,

    def append_bar(self, bar: Dict[str, Any]) -> pd.DataFrame:
        """追加新的 K线数据（增强版：检测并补全缺失）"""
        # 1. 转换为 DataFrame
        new_bar_df = pd.DataFrame([{
            "timestamp": pd.Timestamp.fromtimestamp(bar["timestamp"] / 1000) if isinstance(bar["timestamp"], (int, float)) else pd.Timestamp(bar["timestamp"]),
            "datetime": pd.Timestamp.fromtimestamp(bar["timestamp"] / 1000) if isinstance(bar["timestamp"], (int, float)) else pd.Timestamp(bar["timestamp"]),
            "open": float(bar["open"]),
            "high": float(bar["high"]),
            "low": float(bar["low"]),
            "close": float(bar["close"]),
            "volume": float(bar.get("volume", 0)),
            "symbol": self.symbol,
        }])
        
        # 2. 检测并补全缺失数据
        if self.auto_fill_gaps and self.history_df is not None and len(self.history_df) > 0:
            missing = self._detect_missing_bars(new_bar_df)
            if missing:
                print(f"⚠️ 检测到 {len(missing)} 条缺失数据，开始补全...")
                self._fill_missing_bars(missing)
        
        # 3. 写入 QuestDB
        try:
            self.questdb_writer.write_bar({
                **bar,
                "symbol": self.symbol,
                "timeframe": self.timeframe,
            })
        except Exception as e:
            print(f"⚠️ 写入 QuestDB 失败: {e}")
        
        # 4. 追加到历史数据
        if self.history_df is None:
            self.history_df = new_bar_df
        else:
            # 检查是否重复
            last_timestamp = self.history_df["timestamp"].iloc[-1]
            new_timestamp = new_bar_df["timestamp"].iloc[0]
            
            if new_timestamp > last_timestamp:
                self.history_df = pd.concat([self.history_df, new_bar_df])
                
                # 维护滑动窗口
                max_bars = self.warmup_bars * 2
                if len(self.history_df) > max_bars:
                    self.history_df = self.history_df.tail(max_bars)
            elif new_timestamp == last_timestamp:
                # 相同时间戳，更新（可能是数据修正）
                self.history_df.iloc[-1] = new_bar_df.iloc[0]
            # else: 旧数据，忽略
        
        return self.history_df
    
    def _detect_missing_bars(self, new_bar: pd.DataFrame) -> List[pd.Timestamp]:
        """检测缺失的 K线数据"""
        if len(self.history_df) == 0:
            return []

                if last_timestamp + expected_interval * (i + 1) < new_timestamp - tolerance

    def _detect_missing_in_range(self, df: pd.DataFrame) -> List[pd.Timestamp]:
        """在指定范围内检测缺失数据"""
        if len(df) < 2:
            return []
        
        expected_interval = pd.Timedelta(self.timeframe)
        tolerance = expected_interval * 0.1
        
        df_sorted = df.sort_values("timestamp")
        missing_timestamps = []
        
        for i in range(len(df_sorted) - 1):
            current_time = df_sorted.iloc[i]["timestamp"]
            next_time = df_sorted.iloc[i + 1]["timestamp"]
            
            gap = next_time - current_time
            if gap > expected_interval + tolerance:
                # 计算缺失的时间戳
                missing_count = int((gap - tolerance) / expected_interval)
                for j in range(1, missing_count + 1):
                    missing_time = current_time + expected_interval * j
                    if missing_time < next_time - tolerance:
                        missing_timestamps.append(missing_time)
        
        return missing_timestamps
    
    def manual_fill_gaps(
        self,
        start_time: pd.Timestamp,
        end_time: pd.Timestamp,
    ) -> int:
        """
        手动补全指定时间范围内的缺失数据
        
        Args:
            start_time: 开始时间
            end_time: 结束时间
        
        Returns:
            补全的数据条数
        """
        if not self.gap_filler:
            print("⚠️ 数据补全器未初始化")
            return 0
        
        # 查询 QuestDB 中已有的数据
        missing = self._query_missing_from_questdb(start_time, end_time)
        
        if not missing:
            print("✅ 没有缺失数据")
            return 0
        
        print(f"📥 发现 {len(missing)} 条缺失数据，开始下载...")
        self._fill_missing_bars(missing)
        
        return len(missing)
    
    def _query_missing_from_questdb(
        self,
        start_time: pd.Timestamp,
        end_time: pd.Timestamp,
    ) -> List[pd.Timestamp]:
        """从 QuestDB 查询缺失的时间戳"""
        if not REQUESTS_AVAILABLE:
            return []
        
        # 生成期望的时间序列
        expected_times = pd.date_range(start_time, end_time, freq=self.timeframe)
        
        # 查询 QuestDB 中已有的数据
        query = f"""
        SELECT DISTINCT timestamp
        FROM klines
        WHERE symbol = '{self.symbol}' 
          AND timeframe = '{self.timeframe.replace("T", "m")}'
          AND timestamp >= '{start_time}'
          AND timestamp <= '{end_time}'
        ORDER BY timestamp
        """
        
        try:
            response = requests.post(
                f"{self.questdb_url}/exec",
                data=query,
                timeout=10,
            )
            response.raise_for_status()
            
            import io
            df_db = pd.read_csv(io.StringIO(response.text))
            
            if len(df_db) == 0:
                # 完全没有数据，返回整个时间段
                return expected_times.tolist()
            
            # 找出缺失的时间戳
            existing_times = pd.to_datetime(df_db["timestamp"])
            missing = expected_times.difference(existing_times)
            
            return missing.tolist()
        
        except Exception as e:
            print(f"⚠️ 查询 QuestDB 失败: {e}")
            return []
```

## 五、数据补全策略总结

### 5.1 自动补全（推荐）

**触发时机**：
1. 收到新数据时，检测与前一条数据的间隔
2. 定期检查（如每 60 秒）

**补全方式**：
1. 检测到缺失 → 从交易所 API 下载 → 写入 QuestDB → 插入内存

**优势**：
- 自动化，无需手动干预
- 及时发现和修复数据缺失
- 保证数据完整性

### 5.2 手动补全

**使用场景**：
- 系统重启后，需要补全停机期间的数据
- 发现历史数据有缺失
- 数据质量检查

**使用方法**：
```python
# 补全指定时间范围的数据
data_manager.manual_fill_gaps(
    start_time=pd.Timestamp("2024-01-01 00:00:00"),
    end_time=pd.Timestamp("2024-01-01 23:59:59"),
)
```

### 6.2 注意事项

1. **API 限制**
   - 交易所 API 有频率限制
   - 避免频繁下载，可以批量下载
   - 使用缓存避免重复下载

2. **历史数据范围**
   - 某些交易所只保留最近 N 天的数据
   - 超出范围的数据无法下载
   - 需要定期备份到 Parquet

3. **数据质量**
   - 下载的数据需要验证
   - 检查时间戳、价格、成交量是否合理
   - 过滤异常数据

4. **性能影响**
   - 数据补全可能阻塞主流程
   - 建议异步执行
   - 使用后台任务定期检查

🏷️ 四、标签构建（Label Design）

根据你的交易目标设计标签：
✅ 方案 1：二分类（上涨 vs 不涨）
python
forward_return = close.shift(-n) / close - 1
signal = (forward_return > threshold).astype(int)
n=3 表示预测未来 3 根 K 线的收益
threshold=0.005（0.5%），需大于交易成本
✅ 方案 2：多分类（涨/平/跌）
python
signal = 0
signal[forward_return > 0.005] = 1 # 上涨
signal[forward_return < -0.005] = -1 # 下跌
✅ 方案 3：回归（直接预测收益率）
python
y = forward_return.dropna()

✅ 简单回测逻辑（伪代码）

python
for i in range(warmup, len(df)):
X = df[features].iloc[i:i+1]
pred = model.predict(X)[0]

if pred > 0.5 and not in_position:
buy_price = df['open'].iloc[i+1] # 下一根开盘价买入
position = buy_price (1 - fee)

elif pred <= 0.5 and in_position:
sell_price = df['open'].iloc[i+1]
profit = (sell_price (1 - fee)) / position - 1
total_return *= (1 + profit)
in_position = False

📈 七、绩效评估指标

指标 解释
------ ------
年化收益 (CAGR) 是否跑赢 HODL？
夏普比率 (Sharpe) >1 可接受，>2 优秀
最大回撤 (Max Drawdown) 是否能承受？
胜率 (Win Rate) >50% 较好
盈亏比 (Profit Factor) 总盈利 / 总亏损 >1.5
卡玛比率 (Calmar) CAGR / Max DD >1 为佳
💡 对比基准：买入持有（Buy & Hold）策略

🔁 八、进阶建议：Walk-Forward Analysis（WFA）

不要只训练一次！

✅ 推荐做法：
text
训练：2021–2022 → 测试：2023
训练：2021–2023 → 测试：2024
训练：2021–2024 → 测试：2025

这叫 滚动训练（Rolling Training） 或 WFA，更能检验模型稳定性。

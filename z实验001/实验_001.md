## Evidence 层优化 (Sharpness 目标)

### 为什么用 Sharpness？

Evidence 是**软调整**，不做硬 veto，而是影响置信度/仓位。
Sharpness 衡量特征对好坏样本的区分能力：


```bash
# 运行 Evidence Sharpness 平坦高原优化
python scripts/optimize_evidence_plateau.py \
  --strategy bpc \
  --logs results/train_final_20260207_222521_rr_extreme/bpc/logs_gated.parquet \
  --output results/evidence_optimization.json \
  --min-sharpness 0.05
```

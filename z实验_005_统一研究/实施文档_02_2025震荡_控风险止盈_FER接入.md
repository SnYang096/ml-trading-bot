# 实施文档 02：2025 震荡市控风险、止盈与 FER 接入

## 1. 目标定义

在 2025 震荡窗口中验证：

- 系统从“趋势放大”切换到“风险可控”
- 能主动止盈/减小回撤，不出现持续放大亏损
- FER 在震荡环境接入后，组合收益曲线更平滑并保持正收益能力

## 2. 范围与前提

- 时间范围：`2025-01-01 ~ 2025-12-31`
- 策略组合建议：
  - 趋势腿：`bpc-short-240T`、`me-short-120T`
  - 震荡腿：`fer-short-120T`（可加 `fer-long-120T` 做双边对照）
- 运行方式：`rolling_sim`（必须启用，测试跨月适应）

## 3. 配置策略（建议）

## 3.1 方向与 side policy

- `config/prod_train_pipeline_2h.yaml`
  - `strategy_scope.direction: short`（先做空侧稳定性验证）
  - `symbol_policy.mode: hybrid_carry_forward`
  - `symbol_policy.carry_forward_ttl_months` 设为 1~2（避免长期僵化）

## 3.2 风险与止盈

- 对 ME：保留较宽 `max_stop_pct`（避免被裁成过紧止损）
- 对 FER：禁止加仓，保持反转/回归策略简洁
- 对 BPC：保留趋势骑乘，但观察震荡月是否自动收缩

## 3.3 组合与 slot

- 开启 `pcm_slot_grid`，对比不同 slot 配置在震荡市的稳健性
- 关注 `slot_full_rate`，避免“信号多但都被卡掉”导致机会损失

## 4. 实施步骤

### Step A：先跑滚动模拟（主实验）

```bash
mlbot pipeline run --all \
  --config config/prod_train_pipeline_2h.yaml \
  --stage rolling_sim
```

记录 `run_id`，然后查看：

```bash
mlbot pipeline report-side-state --run-id <run_id> --config config/prod_train_pipeline_2h.yaml
mlbot pipeline debug-quality --run-id <run_id> --month 2025-03 --config config/prod_train_pipeline_2h.yaml
mlbot pipeline debug-quality --run-id <run_id> --month 2025-09 --config config/prod_train_pipeline_2h.yaml
```

### Step B：单独运行 slot 高原对比

```bash
mlbot pipeline run --all \
  --config config/prod_train_pipeline_2h.yaml \
  --stage pcm_slot_grid
```

看 `_pcm_slot_grid/<run_id>/pcm_slot_grid_report.json`。

### Step C：FER 贡献归因

对滚动结果中每月 `pcm.per_archetype` 做统计，关注：

- FER 的月度 `total_r`
- FER 在亏损月是否提供对冲/缓冲

## 5. 验收标准（必须量化）

- 风险控制：
  - 2025 全年拼接 `max_drawdown_r` 显著低于 2024 牛市模式
  - 近止损比例不过度升高（避免“抖动止损”）
- 收益持续：
  - `stitched_total_r > 0`（最低门槛）
  - 月度负收益月可接受，但不应连续失控
- FER 接入效果：
  - 加入 FER 后，组合波动下降或回撤收敛（优先于单纯提高收益）

## 6. 对照实验（建议必须做）

至少做 3 组对照：

1. 不含 FER 的组合
2. 含 FER 的组合
3. 含 FER + 推荐 slot case（来自 `pcm_slot_grid`）

只要第 2/3 组在回撤与稳定性上优于第 1 组，就认为 FER 接入有效。

## 7. 失败诊断优先级

若震荡市仍明显亏损，优先排查：

1. `strategy_scope.direction` 与实际震荡结构不匹配
2. ME 的止损/追踪参数在震荡市仍偏进攻
3. FER 触发样本不足（`n_trades` 太低）
4. slot 配置导致 FER 经常拿不到执行机会

## 8. 上线门槛（本目标）

- 2025 震荡窗口滚动结果通过：
  - 可接受回撤
  - 正收益或轻微正边际
  - FER 对组合稳定性有可复现贡献

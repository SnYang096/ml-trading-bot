# ME 策略训练快速启动

## 命令速查表（复制即用）

### Step 1: 构建 Feature Store

```bash
cd /home/yin/trading/ml_trading_bot

mlbot feature-store build --no-docker \
  --config config/strategies/me \
  --symbols BTCUSDT,ETHUSDT,BNBUSDT,SOLUSDT,XRPUSDT,ADAUSDT \
  --timeframe 60T \
  --start-date 2023-01-01 --end-date 2026-01-01 \
  --warmup-months 6
```

### Step 2: Gate 训练

```bash
# 训练
mlbot train final --no-docker \
  --config config/strategies/me \
  --features config/strategies/me/features_gate.yaml \
  --labels config/strategies/me/labels_rr_extreme.yaml \
  --symbol BTCUSDT,ETHUSDT,BNBUSDT,SOLUSDT,XRPUSDT,ADAUSDT \
  --timeframe 240T --data-path data/parquet_data \
  --start-date 2023-01-01 --end-date 2026-01-01 \
  --holdout-start-date 2024-05-01 --holdout-end-date 2026-01-01 --seed 42

# 记录训练输出目录
GATE_RESULT_DIR="results/train_final_<timestamp>_rr_extreme/me"

# Gate 优化
mlbot gate apply-archetype \
  --logs ${GATE_RESULT_DIR}/predictions.parquet --strategy me

python scripts/optimize_gate_unified.py --strategy me \
  --logs ${GATE_RESULT_DIR}/logs_gated.parquet \
  --output ${GATE_RESULT_DIR}/gate_optimization.json
```

### Step 3: Evidence 训练

```bash
# 训练
mlbot train final --no-docker \
  --config config/strategies/me \
  --features config/strategies/me/features_evidence.yaml \
  --labels config/strategies/me/labels_return_tree.yaml \
  --symbol BTCUSDT,ETHUSDT,BNBUSDT,SOLUSDT,XRPUSDT,ADAUSDT \
  --timeframe 240T --data-path data/parquet_data \
  --start-date 2023-01-01 --end-date 2026-01-01 \
  --holdout-start-date 2024-05-01 --holdout-end-date 2026-01-01 --seed 42

# 记录训练输出目录
EVIDENCE_RESULT_DIR="results/train_final_<timestamp>_return_tree/me"

# Evidence 优化
python scripts/optimize_evidence_plateau.py --strategy me \
  --logs ${EVIDENCE_RESULT_DIR}/logs_gated.parquet \
  --output ${EVIDENCE_RESULT_DIR}/evidence_optimization.json
```

### Step 4: Entry Filter 优化

```bash
python scripts/optimize_entry_filter_snotio.py --all \
  --logs ${EVIDENCE_RESULT_DIR}/predictions.parquet --strategy me

python scripts/optimize_entry_filter_plateau.py \
  --logs ${EVIDENCE_RESULT_DIR}/predictions.parquet --strategy me
```

### Step 5: Execution 优化

```bash
python scripts/optimize_execution_grid.py \
  --logs ${EVIDENCE_RESULT_DIR}/logs_gated.parquet --strategy me \
  --output ${EVIDENCE_RESULT_DIR}/execution_grid.json
```

### Step 6: PCM 联合回测

```bash
# 单独回测 ME
python scripts/backtest_execution_layer.py \
  --logs ${EVIDENCE_RESULT_DIR}/predictions.parquet \
  --strategy me

# ME + BPC PCM 联合回测
python scripts/backtest_execution_layer.py \
  --pcm bpc:results/<bpc_result_dir>/predictions.parquet \
       me:${EVIDENCE_RESULT_DIR}/predictions.parquet \
  --quantile-train-start 2025-02-01 \
  --quantile-train-end 2025-08-01
```

## 检查清单

在执行训练前，确保：

- [ ] 数据已下载：data/parquet_data/ 包含 2023-2026 数据
- [ ] Feature Store 构建完成
- [ ] 每个优化脚本执行后，手动审核输出文件
- [ ] 将优化结果更新到对应的 archetype 配置文件

在每个 Step 完成后：

- [ ] Gate 优化 → 更新 `config/strategies/me/archetypes/gate.yaml`
- [ ] Evidence 优化 → 更新 `config/strategies/me/archetypes/evidence.yaml`
- [ ] Entry Filter 优化 → 更新 `config/strategies/me/archetypes/entry_filters.yaml`
- [ ] Execution 优化 → 更新 `config/strategies/me/archetypes/execution.yaml`
